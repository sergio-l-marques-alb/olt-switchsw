Help for MLD Snooping Multicast Router VLAN Configuration

Selection Criteria

    • Interface - The select box lists all Unit/Slot/Port Slot/Port . Select the interface for which you want Multicast Router to be enabled.

Configurable Data

    • VLAN ID - VLAN ID for which the Multicast Router Mode is to be Enabled or Disabled. Valid range is from 1 to 4093.
    • Multicast Router - For the VLAN ID, multicast router may be enabled or disabled using this.

Command Buttons

    • Submit - Update the switch with the values on the screen. If you want the switch to retain the new values across a power cycle you must perform a save.
