iSCSI Help

Help for iSCSI Targets

Unconfigurable Data

    • TCP Port - TCP port number on which a iSCSI target listens to requests. Upto 16 TCP ports can be defined in the system. iSCSI well-known ports 3260 and 860 are configured as default but can be removed as any other configured target.
    • IP Address - IP address of the iSCSI target.
    • Target Name - iSCSI name of the iSCSI target.

Command Buttons
