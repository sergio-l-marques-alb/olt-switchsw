Help for PIM Global Status

Selection Criteria

    • PIM Protocol Type - The protocol variant of PIM i.e sparse mode or Dense for which admin mode need to be displayed.

Non-Configurable Data

    • Admin Mode - The administrative status of PIM (DM or SM as selected by PIM Protocol Type field) in the router: either enable or disable.
    • Data Threshold Rate - The minimum source data rate in K bits/second above which the last-hop router will switch to a source-specific shortest path tree. This field is applicable for PIMSM only and hence shown on selection PIMSM protocol type.
    • Register Threshold Rate - The minimum source data rate in K bits/second above which the Rendezvous Point router will switch to a source-specific shortest path tree. This field is applicable for PIMSM only and hence shown on selection PIMSM protocol type.
