Help for PIM Interface Configuration

Selection Criteria

    • Interface - Select the interface for which data is to be displayed or configured.

Configurable Data

    • Admin Mode - Select Enable or Disable to set the administrative status of PIM for the specified interface in the router. The default is Disable.
    • Hello Interval (secs) - Enter the time (in seconds) between the Transmission of which PIM Hello messages on this interface. The valid values are from seconds. The default value is seconds.
    • Join/Prune Interval (secs) - Enter the frequency (in seconds) at which PIM Join/Prune messages are transmitted on this PIM interface. This field is applicable for PIMSM only. The valid values are from seconds. The default value is seconds.
    • BSR Border - Select enable or disable to set BSR border status on the selected interface. This field is applicable for PIMSM only.
