Help for IGMP Snooping VLAN Status

Non-Configurable Data

    • VLAN ID - All VLAN IDs for which the IGMP Snooping mode is Enabled.
    • Admin Mode - Igmp Snooping Mode for VLAN ID.
    • Fast Leave Admin Mode - Fast Leave Mode for VLAN ID.
    • Group Membership Interval(secs) - Group Membership Interval of IGMP Snooping for the specified VLAN ID. Valid range is 2 to 3600 seconds. Its value should be greater than maximum response time.
    • Maximum Response Time(secs) - Maximum Response Time of IGMP Snooping for the specified VLAN ID. Valid range is 1 to 25 seconds. Its value should be less than group membership value.
    • Multicast Router Expiry Time(secs) - Multicast Router Expiry Time of IGMP Snooping for the specified VLAN ID. Valid range is 0 to 3600 seconds.

Command Buttons

    • Refresh - Refresh the data on the screen with the present state of the data in the switch.
