DHCP Server Help

Help for DHCP Server Reset Configuration

Selection Criteria

    • Clear - Specifies whether All Dynamic Bindings/Specific Dynamic Binding/All Address Conflicts/Specific Address Conflict is to be deleted.

Configurable Data

    • IP Address - IP Address against the Binding/Address Conflict to be cleared. This field appears only if the user has selected "Specific Dynamic Binding/Specific Address Conflict" in Clear.

Command Buttons

    • Clear All Bindings - Clears/Removes all the dynamic bindings. This is visible when you select All Dynamic Bindings in the Clear selection.
    • Clear Specific Binding - Clears/Removes the specified dynamic binding. This is visible when you select Specific Dynamic Bindings in the Clear selection.
    • Clear All Conflicts - Clears/Removes all the address conflicts. This is visible when you select All Address Conflicts in the Clear selection.
    • Clear Specific Conflict - Clears/Removes the specified address conflict. This is visible when you select Specific Address Conflict in the Clear selection.
