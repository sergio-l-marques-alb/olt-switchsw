DNS Client Help

DNS Hostname - IP Mapping Configuration

Configurable Data

    • Hostname - Enter the host name of the static entry. Host names are composed of series of labels concatenated with dots. Each label must be between 1 and 63 characters long, and the entire host name has a maximum of 255 characters. Host name should start with a letter and end with a letter, digit or dot. Special characters like dot, space, underscore and hyphen are accepted.
    • Inet Address - Enter the IPv4 or IPv6 Address of the static entry.
