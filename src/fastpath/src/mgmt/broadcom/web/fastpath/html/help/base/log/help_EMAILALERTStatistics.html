Help for Email Alert Statistics

Non-Configurable Data

    • Number of Emails Sent - Displays the number of emails sent since last reset.
    • Number of Emails Failed - Displays the number of emails failed since last reset.
    • Time Since Last Email Sent - Time elapsed since the last email sent successfully.
