Help for Transfer Progress

Non-Configurable Data

    • Transfer Status - Display information about the progress of the file transfer.
    • Transfer ModE - Display protocol used to transfer the file.
    • Server IP Address - Display IP Address of the server.
    • File Path - Display file path on server where file is located.
    • File Name - Display the name of the file.
    • Local File Name - Display the name of the local file.
    • Data Type - Display the type of data transfer.
