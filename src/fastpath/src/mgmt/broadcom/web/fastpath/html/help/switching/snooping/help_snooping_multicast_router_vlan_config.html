Help for IGMP Snooping Multicast Router VLAN Configuration

Selection Criteria

    • Interface - The select box lists all interfaces. Select the interface for which you want Multicast Router to be enabled.

Configurable Data

    • VLAN ID - VLAN ID for which the Multicast Router Mode is to be Enabled or Disabled. Valid range is .
    • Multicast Router - For the VLAN ID, multicast router may be enabled or disabled using this. The value is enabled by default.

Command Buttons

    • Submit - Update the switch with the values on the screen. If you want the switch to retain the new values across a power cycle you must perform a save.
