Help for DHCP L2 Relay Subscription Summary

Selection Criteria

    • Interface - The interface for which data is to be displayed.
    • Subscription name - The subscription entry on the selected interface for which data is to be displayed.
    • DHCP L2 Relay Mode - State of the DHCP L2 Relay field on given subscription and given interface shown on the row entry.
    • DHCP L2 Relay Circuit-ID - State of the DHCP Circuit-ID field on given subscription and given interface shown on the row entry.
    • DHCP L2 Relay Remote-ID - State of the DHCP Remote-ID field on given subscription and given interface shown on the row entry.

Command Buttons

    • Refresh - Refresh the data on the screen with the present state of the data in the switch.
