Copy Configuration Files Help

Help for System Utilities Copy Configuration Files

Configurable Data

    • Source File - Select the source configuration file, either running-config, startup-config or backup-config to copy from.
    • Destination File - Select the destination configuration file to copy either to startup-config or to backup-config.

Command Buttons

    • Submit - Update the switch with the values on the screen and to perform the copy operation.
    • Refresh - Refresh the data on the screen with the present state of the data in the switch.
