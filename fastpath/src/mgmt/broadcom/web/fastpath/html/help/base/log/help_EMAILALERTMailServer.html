Help for Email Alert Mail Server Configuration

Configurable Data

    • Mail Server Address - To configure the Mail Server Address. In this release, only one mail server configuration is allowed.
    • Mail Server Security - To configure the Mail Server Security. The possible values are TLSv1 and None.
    • Mail Server User Name - To configure the Mail Server User Name.
    • Mail Server Password - To configure the Mail Server Password.
    • Mail Server Port - To configure the Mail Server Port.
    • Remove - Check Box to delete the selected Mail Server Entry.

      • Non-Configurable Data

        • Mail Server Address - Displays the Mail Server Address.
        • Mail Server Security - Displays the Mail Server Security. The possible values are TLSv1 and None.
        • Mail Server User Name - Displays the Mail Server User Name.
        • Mail Server Password - Displays the Mail Server Password.
        • Mail Server Port - Displays the Mail Server Port.
