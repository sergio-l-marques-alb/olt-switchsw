Help for DHCP L2 Relay Interface Statistics

Selection Criteria

    • Interface - Select the DHCP L2 Relay enabled interface for which statistics to be displayed.

Non-Configurable Data

    • Untrusted Server Messages with Option-82 - Number of DHCP Reply packets received with Option-82 on untrusted DHCP L2 Relay interface.
    • Untrusted Client Messages with Option-82 - Number of DHCP Request packets received with Option-82 on untrusted DHCP L2 Relay interface.
    • Trusted Server Messages without Option-82 - Number of DHCP Reply packets received without Option-82 on untrusted DHCP L2 Relay interface.
    • Trusted Client Messages without Option-82 - Number of DHCP Request packets received without Option-82 on untrusted DHCP L2 Relay interface.

Command Buttons

    • Refresh - Refreshes the data on the screen with the present state of the data in the switch.
    • Clear - Clears statistics for the selected interface.
    • ClearAll - Clears statistics for all the interfaces.
