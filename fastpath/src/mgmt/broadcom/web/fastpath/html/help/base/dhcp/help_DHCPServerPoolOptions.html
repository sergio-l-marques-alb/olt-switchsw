DHCP Server Help

Help for DHCP Server Pool Options

Selection Criteria

    • Pool Name - Shows all the existing Pool Names.

Non-Configurable Data

    • Option Code - Specifies the Option Code configured for the selected Pool.
    • ASCII Value - Specifies the Option ASCII Value for the selected pool.
    • Hex Value - Specifies the Option Hex Value for the selected pool.
    • IP Address Value - Specifies the Option IP Address Value for the selected pool.

Command Buttons

    • Delete - Deletes the checked option for the selected Pool.
