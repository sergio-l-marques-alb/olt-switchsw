Help for Email Alert Global Configuration

Configurable Data

    • Admin Mode - To enable or disable email alerts to SMTP Server. Setting this to disable stops emailing to SMTP Servers.
    • From Address - Configures the email alert from address.
    • Log Duration (min) - This duration in minutes determines how frequently the non critical messages are sent to the SMTP Server.
    • Urgent Messages Severity - Configures the severity level for urgent log messages. The severity level options are enumerated below:
      • Option 'Emergency' indicates system is unusable. It is the highest level of severity.
      • Option 'Alert indicates action must be taken immediately
      • Option 'Critical indicates critical conditions
      • Option 'Error indicates error conditions
      • Option 'Warning indicates warning conditions
      • Option 'Notice indicates normal but significant conditions
      • Option 'Informational' indicates informational messages
      • Option 'Debug' indicates debug-level messages
      By default it is Alert.
    • Non Urgent Messages Severity - Configures the severity level for non-urgent log messages. The severity level options are enumerated below:
      • Option 'Emergency' indicates system is unusable. It is the highest level of severity.
      • Option 'Alert indicates action must be taken immediately
      • Option 'Critical indicates critical conditions
      • Option 'Error indicates error conditions
      • Option 'Warning indicates warning conditions
      • Option 'Notice indicates normal but significant conditions
      • Option 'Informational' indicates informational messages
      • Option 'Debug' indicates debug-level messages
      By default it is Warning.
    • Traps Severity - Configures the severity level for trap log messages. The severity level options are enumerated below:
      • Option 'Emergency' indicates system is unusable. It is the highest level of severity.
      • Option 'Alert indicates action must be taken immediately
      • Option 'Critical indicates critical conditions
      • Option 'Error indicates error conditions
      • Option 'Warning indicates warning conditions
      • Option 'Notice indicates normal but significant conditions
      • Option 'Informational' indicates informational messages
      • Option 'Debug' indicates debug-level messages
      By default it is Informational.
    • Test Message Type - Configures the message type for testing email alert functionality.
    • Test Message Body - Configures the message body for testing email alert functionality.
