Help for Router Discovery Configuration

Selection Criteria

    • Interface - Select the router interface for which data is to be configured.

Configurable Data

    • Advertise Mode - Select enable or disable from the pull-down menu. If you select enable, Router Advertisements will be transmitted from the selected interface.The Default value is .
    • Advertise Address - Enter the IP Address to be used to advertise the router. Allowed addresses are '224.0.0.1' and '255.255.255.255'. Default Address is '224.0.0.1'.
    • Maximum Advertise Interval (secs) - Enter the maximum time (in seconds) allowed between router advertisements sent from the interface.It can be an integer from seconds. The default time is seconds.
    • Minimum Advertise Interval(secs) - Enter the minimum time (in seconds) allowed between router advertisements sent from the interface.It can be an integer from 3 to Maximum Advertise Interval. The default time is seconds.
    • Advertise Lifetime (secs) - Enter the value (in seconds) to be used as the lifetime field in router advertisements sent from the interface. This is the maximum length of time that the advertised addresses are to be considered as valid router addresses by hosts.It can be an integer from Maximum Advertise Interval to . The default time is seconds.
