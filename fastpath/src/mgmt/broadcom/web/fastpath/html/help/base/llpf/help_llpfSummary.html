Help for LLPF Interface Summary

Non-Configurable Data

    • Interface - The physical or LAG interface for which data is to be displayed.
    • Blocked Protocols -
        The Configured (Enabled) Block protocol Type for a particular interface. Following are the blocking protocols supported:
      • ISDP- Industry Standard Discovery Protocol
      • VTP- VLAN Trunking Protocol
      • DTP- Dynamic Trunking Protocol
      • UDLD- Uni Directional Link Detection
      • PAGP- Port Aggregation Protocol
      • SSTP- Shared Spanning Tree Protocol
      • Blockall- All the above mentioned protocols will be dropped in addition to protocols with a Destination MAC of "01:00:0C:CC:CC:CX".

Command Buttons

    • Refresh - Refreshes the data on the screen with the present state of the data in the switch.
