Help for Multiple Port Mirroring - Remove Source Ports

Configurable Data

    • Session - Specifies the selected Session ID for which source ports are to be removed.
    • Source Port(s) - Specifies the configured port(s) with directions as mirrored port(s). Traffic of the configured port(s) is sent to the probe port.
    • Source Port(s) - Specifies the configured port(s) as mirrored port(s). Traffic of the configured source port(s) is sent to the probe port.

Non-Configurable Data

  • Direction - Specifies the direction of traffic on source port(s) which will be sent to the probe port. Possible values are :
    • Tx and Rx - Both Ingress and Egress traffic.
    • Rx - Ingress traffic only.
    • Tx - Egress traffic only.
  • Source Port(s) - Specifies the configured port(s) with directions as mirrored port(s). Traffic of the configured port(s) is sent to the probe port.

    • Command Buttons

      • Remove - Removes the selected ports from the list of configured source ports.
