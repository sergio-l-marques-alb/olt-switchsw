Help for BOOTP/DHCP Relay Agent Configuration

Configurable Data

Non-Configurable Data
    • Maximum Hop Count - The maximum number of hops a client request can take before being discarded. Enter within range 1-16. Default is 4.
    • Minimum Wait Time - The Minimum time in seconds. This value will be compared to the time stamp in the client's request packets, which should represent the time since the client was powered up. Packets will only be forwarded when the time stamp exceeds the minimum wait time.The valid range is from seconds. The default value is seconds.
    • Circuit ID Option Mode - This is the Relay agent option which can be either enable or disable. When 'Enable' Relay Agent options will be added to requests before they are forwarded to the server and removed from replies before they are forwarded to clients.The default value is 'Disable'.

Note:Configuring/Viewing of BOOTP/DHCP Relay Agent Address in global mode is deprecated! To achieve this functionality use Helper IP Configuration pages.
