Help for IP Helper Interface Configuration

Selection Criteria

    • Source IP Interface - Select the interface from the pull down menu to get the relay entries configured on a particular interface. If "All" is selected all the configured relay entries on all interfaces will be displayed.

Configurable Data

    • Remove - Removes the specified UDP Relay Entry when selected and "Submit" is pressed.

Non-Configurable Data

    • Source IP Interface - The interface for which the relay entry is configured.
    • UDP Destination Port - The destination UDP port ID number of UDP packets to be relayed.
    • Server Address - The IPv4 address of the server to which packets are relayed for the specific UDP Destination Port.
    • IsDiscard - If True, packets arriving on the given interface with the given destination UDP port are discarded rather than relayed. Discard entries are used to override global IP helper address entries which otherwise might apply to a packet.
    • Hit Count - The number of times a packet has been forwarded or discarded according to this entry.
