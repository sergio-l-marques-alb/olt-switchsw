Help for IP DSCP

Selection Criteria

    • DSCP Keyword - This lists the keywords for the known DSCP values from which one can be selected. The list includes 'other' as an option for the remaining values.

Configurable Data

    • DSCP Value - This is a DiffServ Code Point value in the range 0 to 63 to compare against. This field is only displayed when the DSCP Keyword field is specified as 'other'.
    • Exclude - This is a checkbox that specifies the inclusion or exclusion of the configured match criterion. If the match criterion is to be excluded, this box is checked. Conversely, if the match criterion is to be included, this box is left unchecked.
      •

Non-Configurable Data

    • Class Name - Displays name of the specified DiffServ class.
    • Class Type - Displays type of the specified class.
    • Class Layer 3 Protocol - Displays type of layer 3 protocol of the specified Diffserv class.
