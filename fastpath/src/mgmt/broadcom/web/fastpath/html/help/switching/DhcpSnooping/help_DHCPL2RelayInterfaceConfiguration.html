Help for DHCP L2 Relay Interface Configuration

Selection Criteria

    • Interface - Selects the interface for which data is to be displayed or configured.

Configurable Data

    • DHCP L2 Relay Mode - Enables or Disables L2 Relay mode on selected interface. The factory default is Disable.
    • DHCP L2 Relay Trust Mode - If this is Enabled, DHCP L2 Relay application considers selected port as L2 Relay trusted. The factory default is Disable.

Non-configurable data

    • DHCP L2 Relay Mode - Displays L2 Relay mode on selected interface.
    • DHCP L2 Relay Trust Mode - Displays if DHCP L2 Relay application considers the selected port as L2 Relay trusted.

Command Buttons

    • Submit - Update the switch with the values on the screen. If you want the switch to retain the new values across a power cycle you must perform a save.
    • Refresh - Refresh the data on the screen with the present state of the data in the switch.
