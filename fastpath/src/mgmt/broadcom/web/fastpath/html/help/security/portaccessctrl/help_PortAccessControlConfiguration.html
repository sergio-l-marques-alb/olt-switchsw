Help for Port Access Control Configuration

Selection Criteria

    • Port - the port to be configured. When the selection is changed, a screen refresh will occur causing all fields to be updated for the newly selected port. All physical interfaces are valid.

Configurable Data

    • Pae capabilities - This selector lists the options for Port access entity (PAE) configuration. The options are:
      • authenticator : Port access entity (PAE) is set to Authenticator.
      • supplicant : Port access entity (PAE) is set to Supplicant.
