Help for Email Alert To Address Configuration

Configurable Data

    • Message Type - To configure the logging email alert message type.
    • To Address - To configure the logging email alert to address. Maximum of five addresses can be configured per message type.
    • Remove - Check Box to delete the selected email alert to address entry.

      • Non-Configurable Data

        • Message Type - Displays the logging email alert message type. Valid values are Urgent and Non-Urgent.
        • To Address - Displays the logging email alert to address.
