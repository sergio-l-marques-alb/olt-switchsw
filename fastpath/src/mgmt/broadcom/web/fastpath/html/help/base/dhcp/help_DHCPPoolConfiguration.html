Configurable Data

Non-Configurable Data
    • Pool Name* - For a user with readwrite permission, this field would show names of all the existing pools along with an additional option "Create". When the user selects "Create" another text box "Pool Name" appears where the user may enter name for the Pool to be created. For a user with readonly permission, this field would show names of the existing pools only.
    • Pool Name - This field appears when the user with read-write permission has selected "Create" in the Drop Down list against Pool Name*. Specifies the Name of the Pool to be created. Pool Name can be upto characters in length.
    • Type of Binding - Specifies the type of binding for the pool.
      • Unallocated
      • Automatic
      • Manual
    • Network Number - Specifies the subnet number for a DHCP address of a automatic pool. This field is displayed upon the selection of Automatic binding
    • Network Mask - Specifies the subnet number for a DHCP address of a automatic pool. Either Network Mask or Prefix Length can be configured to specify the subnet mask but not both. This field is displayed upon the selection of Automatic binding
    • Prefix Length - Specifies the subnet number for a DHCP address of a automatic pool. Either Network Mask or Prefix Length can be configured to specify the subnet mask but not both. This field is displayed upon the selection of Automatic binding. Valid Range is
      • .
    • Prefix Length Option Enable - Checkbox provides option to configure the Network Number using Prefix Length or Network Mask. Network Mask and Prefix Length are mutually exclusive. This field is displayed upon the selection of Automatic binding.
    • Client Name - Specifies the Client Name for DHCP manual Pool. This field is displayed upon the selection of Manual binding.
    • Hardware Address - Specifies the MAC address of the hardware platform of the DHCP client. This field is displayed upon the selection of Manual binding.
    • Hardware Address Type - Specifies the protocol of the hardware platform of the DHCP client. Valid types are Ethernet and IEEE802. Default value is Ethernet. This field is displayed upon the selection of Manual binding.
    • Client ID - Specifies the Client Identifier for DHCP manual Pool. This field is displayed upon the selection of Manual binding. In some systems, such as Microsoft DHCP clients, the client identifier is required instead of hardware address.
    • Host Number - Specifies the IP address for a manual binding to a DHCP client. Host can be set only if at least one among of Client Identifier or Hardware Address is specified. Deleting Host would delete Client Name, Client ID, Hardware Address for the Manual Pool and set the Pool Type to Unallocated. This field is displayed upon the selection of Manual binding.
    • Host Mask - Specifies the subnet mask for a manual binding to a DHCP client. Either Host Mask or Prefix Length can be configured to specify the subnet mask but not both. This field is displayed upon the selection of Manual binding.
    • Prefix Length - Specifies the subnet mask for a manual binding to a DHCP client. Either Host Mask or Prefix Length can be configured to specify the subnet mask but not both. This field is displayed upon the selection of Manual binding. Valid Range is
      • .
    • Prefix Length Option Enable - Checkbox provides option to configure the Host Number using Prefix Length or Host Mask. Host Mask and Prefix Length are mutually exclusive. This field is displayed upon the selection of Manual binding.
    • Lease Time - Can be selected as "Infinite" to specify lease time as Infinite or "Specified Duration" to enter a specific lease period. In case of automatic binding infinite implies a lease period of 60 days and In case of manual binding infinite implies indefinite lease period. Default Value is "Specified Duration".
    • Days - Specifies the Number of Days of Lease Period. This field appears only if the user has specified "Specified Duration" as the Lease time. Default value is 1. Valid Range is .
    • Hours - Specifies the Number of Hours of Lease Period. This field appears only if the user has specified "Specified Duration" as the Lease time. Valid Range is
      • . Default value is 0.
    • Minutes - Specifies the Number of Minutes of Lease Period. This field appears only if the user has specified "Specified Duration" as the Lease time. Valid Range is
      • . Default value is 0.
    • Default Router Addresses - Specifies the list of Default Router Addresses for the pool. The user may specify upto 8 Default Router Addresses in order of preference.
    • DNS Server Addresses - Specifies the list of DNS Server Addresses for the pool. The user may specify upto 8 DNS Server Addresses in order of preference.
    • NetBIOS Name Server Addresses - Specifies the list of NetBIOS Name Server Addresses for the pool. The user may specify upto 8 NetBIOS Name Server Addresses in order of preference.
    • NetBIOS Node Type - Specifies the NetBIOS node type for DHCP clients.
      • b-node Broadcast
      • p-node Peer-to-Peer
      • m-node Mixed
      • h-node Hybrid
    • Next Server Address - Specifies the Next Server Address for the pool.
    • Domain Name - Specifies the domain name for a DHCP client. Domain Name can be upto characters in length.
    • Boot File - Specifies the name of the default boot image for a DHCP client. File Name can be upto characters in length.
    • Add Option - This field is used to configure the DHCP server options.
    • Code - Specifies the DHCP option code. Valid Range is
    • ASCII Value - Specifies an NVT ASCII character string.
    • Hex Value - Specifies hexadecimal data. Each byte in hexadecimal character strings is 2 hexadecimal digits. Each byte can be separated by a colon or white space. A period can be used to separate 2 bytes/4 hexadecimal digits.
    • IP Address Value - Specifies the Option IP addresses.
