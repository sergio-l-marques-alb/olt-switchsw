Help for DHCP Client Parameters

This page displays the DHCP Lease information for the selected IP Interface.

Selection Criteria

    • Interface - Select the interface for which data is to be displayed.

Non-Configurable Data

    • IP Address - The IP Address leased by the DHCP Server.
    • Subnet Mask - The Subnet mask of the IP Address leased by the DHCP Server.
    • DHCP Server Address - The IP Address of the DHCP Server that leased the IP Address.
    • Client State - The State of the DHCP Client on this interface.
    • Transaction ID - The transaction ID of the DHCP Client.
    • Lease Time - The time remaining since the IP address is leased by the DHCP Server.
    • Renewal Time - The time remaining to send the next DHCP Renew Request by DHCP Client to renew the leased IP address.
    • Rebind Time - The time remaining to start the DHCP Rebind process.
    • Retry Count - The number of times the DHCP Client sends a DHCP REQUEST message before the server responds.

Command Buttons

    • Back - Turns the control back to IP Interface Configuration Page.
    • Refresh - Refreshes the data on the screen with the present state of the data in the switch.
