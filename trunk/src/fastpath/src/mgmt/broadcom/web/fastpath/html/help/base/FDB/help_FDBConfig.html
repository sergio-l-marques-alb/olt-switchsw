Help for Forwarding Database Configuration


You use this panel to set the Address Aging Timeout for the forwarding database.

Configurable Data

    • Aging Interval (secs) - The forwarding database contains static entries, which are never aged out, and dynamically learned entries, which are removed if they are not updated within a given time. You specify that time by entering a value for the Address Aging Timeout. You may enter any number of seconds between and .
      • IEEE 802.1D recommends a default of 300 seconds, which is the factory default.
