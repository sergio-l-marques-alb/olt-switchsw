Help for Email Alert Subject Configuration

Configurable Data

    • Message Type - To configure the logging email alert message type.
    • Email Subject - To configure the logging email alert subject.Only alpha numeric characters are allowed for subject
    • Remove - Check Box to replace the selected email alert subject with its default value.

      • Non-Configurable Data

        • Message Type - Displays the logging email alert message type. Valid values are Urgent and Non-Urgent.
        • Email Subject - Displays the logging email alert subject.
