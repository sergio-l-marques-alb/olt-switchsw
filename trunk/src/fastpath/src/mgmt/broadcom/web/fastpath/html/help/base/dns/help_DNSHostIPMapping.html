HostName IP Mapping Summary Help

Help for DNS HostName IP Mapping Summary

Configurable Data

      DNS Static Entries

    • Remove Static - Removes the selected Host Name IP Mapping entry from the Static Host Name IP Mapping list.

      DNS Dynamic Entries

    • Remove Dynamic - Removes the selected Host Name IP Mapping entry from the Dynamic Host Name IP Mapping list.

Non-Configurable Data

      DNS Static Entries

    • Host Name - The Host Name of the static entry.
    • Inet Address - The IPv4 or IPv6 Address of the static entry.

      DNS Dynamic Entries

    • Host Name - The Host Name of the dynamic entry.
    • Total - Total time of the dynamic entry.
    • Elapsed - Elapsed time of the dynamic entry.
    • Type - The type of the dynamic entry.
    • Addresses - The Ipv4 or Ipv6 Address of the dynamic entries.

Command Buttons

    • Add Static Entry - Loads Host Name IP Mapping configuration page to configure Host Name IP mapping entries.
