Help for VRRP Interface Tracking Summary

Selection Criteria

    • emweb:/USMWEB_UNIT_SLOT_PORT_DISPLAY; - The interface for which data is to be displayed.
    • Virtual Router ID - The Virtual Router ID for which data is to be displayed.

Non-Configurable Data

    • Tracking Interface - The Tracked interface for which data is to be displayed.
    • Priority Decrement - The priority decrement for the tracked interface.
    • Interface State - The IP state of the tracked interface.

Command Buttons

    • Refresh - Refresh the data on the screen with the present state of the data in the switch.
