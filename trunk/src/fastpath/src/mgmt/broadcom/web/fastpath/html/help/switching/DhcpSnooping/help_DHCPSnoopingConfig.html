DHCP Snooping Help

Help for DHCP Snooping Configuration

Configurable Data

    • DHCP Snooping Mode - Enables or Disables the DHCP Snooping feature. The factory default is Disable.
    • MAC Address Validation - Enables or Disables the validation of sender MAC Address for DHCP Snooping. The factory default is Enable.

Non-configurable data

    • DHCP Snooping Mode - Displays the current administration mode for the DHCP Snooping feature.
      • The factory default is Disable.
    • MAC Address Validation - Displays the current administration mode for the DHCP Snooping MAC Address Validation. The factory default is Enable.

Command Buttons

    • Submit - Update the switch with the values on the screen. If you want the switch to retain the new values across a power cycle you must perform a save.
