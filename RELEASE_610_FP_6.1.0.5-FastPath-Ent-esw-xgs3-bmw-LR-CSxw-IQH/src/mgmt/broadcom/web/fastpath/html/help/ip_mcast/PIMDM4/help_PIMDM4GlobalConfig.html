Protocol Independent Multicasting (PIM) Protocol Help

PIM-DM Global Configuration

Configurable Data

    • Admin Mode - Select enable or disable from the pulldown menu to set the administrative status of PIM-DM in the router. The default is disable.

Command Buttons

    • Submit - Send the updated configuration to the router. Configuration changes take effect immediately. These changes will not be retained across a power cycle unless a save is performed.
