Help for MLD Snooping Multicast Router Status

Selection Criteria

    • Unit/Slot/Port Slot/Port - The single select box lists all physical and LAG interfaces. Select the interface for which you want to display the status.
    • Slot/Port - The single select box lists all physical and LAG interfaces. Select the interface for which you want to display the status.

Non-Configurable Data

    • Multicast Router - Specifies for the selected interface whether multicast router is enable or disabled.

Command Buttons

    • Refresh - Fetches the database and displays it again starting with the first entry in the table.
