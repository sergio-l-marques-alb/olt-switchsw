Help for DHCP L2 Relay VLAN Configuration

Selection Criteria

    • VLAN ID - Selects the VLAN for which data is to be displayed or configured.

Configurable Data

    • DHCP L2 Relay Mode - Enables or disables the DHCP L2 Relay feature on selected VLAN. The factory default is Disable.
    • DHCP L2 Relay Circuit-Id - Enables or disables the DHCP Circuit-Identifier feature on selected VLAN. The factory default is Disable.
    • DHCP L2 Relay Remote-Id - Sets or Resets the DHCP Remote-Identifier string on selected VLAN. The factory default is NULL string. Range is (0 to 33 characters).

Non-configurable data

    • DHCP L2 Relay Mode - Displays the DHCP L2 Relay feature mode on selected VLAN.
    • DHCP L2 Relay Circuit-Id - Displays the DHCP Circuit-Identifier feature on selected VLAN.
    • DHCP L2 Relay Remote-Id - Displays the DHCP Remote-Identifier string on selected VLAN.

Command Buttons

    • Submit - Applies the new configuration and causes the changes to take effect. These changes will not be retained across a power cycle unless a save configuration is performed.
    • Refresh - Refresh the data on the screen with the present state of the data in the switch.
