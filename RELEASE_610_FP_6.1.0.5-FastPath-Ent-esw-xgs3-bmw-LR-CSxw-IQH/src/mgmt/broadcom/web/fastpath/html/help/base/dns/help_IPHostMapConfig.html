DNS Client Help

DNS Hostname - IP Mapping Configuration

Configurable Data

    • Hostname - Enter the host name of the static entry.
    • Inet Address - Enter the IPv4 or IPv6 Address of the static entry.

Command Buttons

    • Submit - Send the updated configuration to the router. Configuration changes take effect immediately. These changes will not be retained across a power cycle unless a save is performed.
    • Back - Loads back hostname ip mapping page to see configured hostname-ip mapping entries.
