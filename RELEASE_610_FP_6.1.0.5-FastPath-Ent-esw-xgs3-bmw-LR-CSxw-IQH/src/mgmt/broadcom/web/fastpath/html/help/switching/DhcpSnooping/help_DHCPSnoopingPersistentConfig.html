DHCP Snooping Help

Help for DHCP Snooping Persistent Configuration

Selection Criteria

    • Store - Specifies whether on want to store snooping database locally or remotely.
      • Local - Check the Local Checkbox to disable the Remote objects like Remote File Name and Remote IP Address.
      • Remote - Check the Remote Checkbox to Enable the Remote objects like Remote File Name and Remote IP Address.

Configurable Data

    • Remote IP Address - Configures Remote IP Address on which the snooping database will be stored when Remote checkbox is selected.
    • Remote File Name - Configures Remote file name to store the database when Remote checkbox is selected. Range is (1 to 32 Characters).
    • Write Delay - Configures the maximum write time to write the database into local or remote. The range of Write Delay is .

Non-configurable data

    • Remote IP Address - Displays Remote IP Address on which the snooping database will be stored. If not configured, the configuration is stored locally.
    • Remote File Name - Displays Remote file name to store the database. If not configured, the configuration is stored locally.
    • Write Delay(sec) - Displays the maximum write time to store the database locally or remotely. The range of Burst Interval is .

Command Buttons

    • Submit - Applies the new configuration and causes the changes to take effect. These changes will not be retained across a power cycle unless a save configuration is performed.
