Help for MLD Snooping Multicast Router Configuration

Selection Criteria

    • Unit/Slot/Port Slot/Port - The select box lists all Unit/Slot/Port Slot/Port . Select the interface for which you want Multicast Router to be enabled .
    • Multicast Router Mode - Enable or disable Multicast Router on the selected Unit/Slot/Port Slot/Port .

Command Buttons

    • Submit - Updates the switch with the values you entered.
