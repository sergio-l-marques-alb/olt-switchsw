Command Logger Configuration Help

Help for Log Command Logger Configuration

Configurable Data

    • Admin Mode - Enable/Disable the operation of the CLI Command logging by selecting the corresponding pull down field and clicking Submit. By default it is Disable.
    • Admin Status - Disable prevents CLI commmands from being logged. Enable will result in all CLI commands being logged. By default it is Disable.

Command Buttons

    • Submit - Updates the switch with the values you entered.
