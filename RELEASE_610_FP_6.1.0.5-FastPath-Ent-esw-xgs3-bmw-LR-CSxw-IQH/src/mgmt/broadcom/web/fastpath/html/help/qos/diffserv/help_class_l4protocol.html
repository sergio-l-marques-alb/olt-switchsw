Help for Source Layer 4 Port

Selection Criteria

    • Port Keyword - This lists the keywords for the known source layer 4 ports from which one can be selected. The list includes 'other' as an option for the unnamed ports.

Configurable Data

    • Start Port Number - This is the starting port number and is an integer in the range of 0 to 65535 on platforms that support layer 4 port ranges. This field is only displayed when the Port Keyword field is specified as 'other'.
    • End Port Number - This is the ending port number and is an integer in the range of 0 to 65535 on platforms that support layer 4 port ranges. This field is only displayed when the Port Keyword field is specified as 'other'.
    • Port Number - This is an integer in the range of 0 to 65535 on platforms that do not support layer 4 port ranges. This field is only displayed when the Port Keyword field is specified as 'other'.
      •
    • Exclude - This is a checkbox that specifies the inclusion or exclusion of the configured match criterion. If the match criterion is to be excluded, this box is checked. Conversely, if the match criterion is to be included, this box is left unchecked.
      •

Non-Configurable Data

    • Class Name - Displays name of the specified DiffServ class.
    • Class Type - Displays type of the specified class. Class types are platform dependent.
    • Class Layer 3 Protocol - Displays type of layer 3 protocol of the specified Diffserv class.

Command Buttons

    • Submit - Send the updated screen to the switch and cause the changes to take effect on the switch but these changes will not be retained across a power cycle unless a save is performed.
    • Cancel - Cancel the currently selected filter.
