Help For OSPF Neighbor Table

This screen shows the OSPF Neighbor Table, either for all interfaces on which valid OSPF Neighbors are present or the neighbors specific to a given interface on which OSPF Neighbors exist. This information is displayed only if OSPF is enabled and there exists at least on OSPF enabled interface having a valid neighbor.

Non-Configurable Data

    • Router ID - A 32-bit integer in dotted decimal format representing the Router ID of the neighbor on the selected Interface.
    • IP Address - The IP Address of the neighbor on the selected Interface.
    • Neighbor Interface Index - A Unit/Slot/Port Slot/Port identifying the neighbor interface index.

Command Buttons

    • Refresh - Refreshes the page with the latest OSPF neighbor information for the selected interface.
