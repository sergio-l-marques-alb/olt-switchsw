Help for System Utilities Dual Image Configuration


The Dual Image feature allows switch to retain two Switch/System images in permanent storage. The user designates one of these images as the active image to be loaded during subsequent switch restarts. This feature reduces switch down time when upgrading / downgrading the Switch/System software.


The Dual Image feature allows switch to retain two FASTPATH images in permanent storage. The user designates one of these images as the active image to be loaded during subsequent switch restarts. This feature reduces switch down time when upgrading / downgrading the FASTPATH software.

Selection-Criteria

    • Unit - Select the unit whose code image you want to activate, update, or delete.
    • Image Name - Select one of the images from the list.

Configurable Data

    • Image Description - Specify the description for the image that you have selected.

Non-Configurable Data

    • Current-active - This displays the currently active image for the selected unit.

Command Buttons

    • Activate - Makes the selected image the next active image for subsequent reboots.
    • Delete - Deletes the selected image from permanent storage on the switch.
    • Update Bootcode - Updates the bootloader with the selected image.
    • Submit - Updates image description on the switch.
