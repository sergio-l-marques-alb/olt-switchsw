Help for Differentiated Services Service Summary

Non-Configurable Data

    • Unit/Slot/Port Slot/Port - Shows the Unit/Slot/Port Slot/Port that uniquely specifies an interface.
    • Direction - Shows traffic direction for this service interface.
    • Oper. Status - Shows the operational status of this service interface, either Up or Down.
    • Policy Name - Shows the name of the attached policy.

Command Buttons

    • Refresh - Refresh the displayed data.
