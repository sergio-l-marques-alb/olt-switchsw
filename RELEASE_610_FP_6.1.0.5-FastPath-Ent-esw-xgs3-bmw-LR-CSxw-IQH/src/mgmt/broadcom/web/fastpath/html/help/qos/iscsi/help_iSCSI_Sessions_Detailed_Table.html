iSCSI Help

Help for iSCSI Sessions Detailed

Non-Configurable Data

    • Session Index - This displays the list of Session indices. Rest of the fields in the page correspond to current selected Session Index.
    • Target Name - This displays the Target's name.
    • Initiator Name - This displays the Initiator's name.
    • Up Time - This displays the time elapsed since when the current session got created.
    • Time for aging out (in Seconds) - This displays the time left for the current session to expire in time units of seconds.
    • ISID (Initiator Session ID) - The unique identifier an initiator assigns to its session endpoint, which when combined with the iSCSI initiator name provides a unique name in the world for the SCSI initiator port.
    • Initiator IP Address - This displays the Initiator IP Address.
    • Initiator TCP Port - This displays the Initiator TCP Port number of one of the connection's between the Target and initiator.
    • Target IP Address - This displays the IP Address of the Target.
    • Target TCP Port - This displays the Target TCP Port number of one of the connection's between the Target and Initiator.

Command Buttons

    • Refresh - Refresh the data on the screen with the present state of the data in the switch.
