Tunnel Summary Help

Help for Tunnel Summary

This page displays a summary of the configured tunnels.

Non-Configurable Data

Tunnels Table

    • Tunnel ID - The Tunnel ID.
    • Tunnel Mode - The corresponding mode of the Tunnel.
    • IPv6 Mode - IPv6 Enabled/Disabled on the tunnel.
    • Source - The corresponding Tunnel Source Address. In the case where an interface has been configured both the interface and the address are displayed. If the source interface has no address configured the text 'unconfigured' is displayed in place of the address. A 0.0.0.0 address indicates that the Source is Address Type but does not have an IP address configured.
    • Destination - The corresponding Tunnel Destination Address. A 0.0.0.0 address indicates that the Destination address is not configured.
    • IPv6 Unreachables - ICMPv6 Unreachable message generation configuration on the tunnel.

IPv6 Address Table

    • Tunnel ID - The Tunnel ID.
    • IPv6 Address - IPv6 Address configured on the tunnel.
    • IPv6 Address State - Current operational state of the IPv6 Address.

Command Buttons

    • Refresh - Refresh the page with the latest Tunnel entries.
