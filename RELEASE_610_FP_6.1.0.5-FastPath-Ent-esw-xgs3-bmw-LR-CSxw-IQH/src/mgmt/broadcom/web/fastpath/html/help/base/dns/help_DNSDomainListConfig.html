Help for DNS Domain List Configuration

Configurable Data

    • Domain Name - Enter a domain name to complete unqualified names. Domain names are composed of series of labels concatenated with dots. Each label must be between 1 and 63 characters long, and the entire domain name has a maximum of 255 characters.The list is empty by default.

Command Buttons

    • Submit - Send the updated configuration to the router. Configuration changes take effect immediately. These changes will not be retained across a power cycle unless a save is performed.
    • Back - Loads global configuration page back to see configured domain lists.
