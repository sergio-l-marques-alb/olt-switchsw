Global Configuration Help

Help for DNS Global Configuration

Configurable Data

    • Admin Mode - Select enable or disable from the pull down menu to set the administrative status of DNS Client. The default is enable.
    • Default Domain Name - Enter the default domain name for DNS Client. The length of the name should not be longer than 255 characters. By default no default domain-name is configured in the system.
    • Retry Number - Enter the number of times to retry sending DNS queries. The valid values are from 0 to 100. The default value is 2.
    • Response Timeout - Enter the amount of time to wait for a response to a DNS query. The valid values are from 0 to 3600. The default value is 3.

Non-Configurable Data

    • Domain List - The domain name list for DNS Client.
    • Remove - Removes the selected domain name entry from the domain name list.

Non-Configurable Data

    • DNS Client Admin Mode - The administrative status of DNS Client. The default is disable.
    • Default Domain Name - The default domain name for DNS Client. By default no default domain-name is configured in the system.
    • Retry Number - The number of times to retry sending DNS queries. The default value is 2.
    • Response Timeout - The amount of time to wait for a response to a DNS query. The default value is 3.
    • Domain List - The domain name list for DNS Client.

Command Buttons

    • Submit - Update the switch with the values on the screen. If you want the switch to retain the new values across a power cycle you must perform a save.
    • Create - Loads domain name list configuration page to configure domain name list.
    • Delete - Deletes all the entries selected using Remove option.
    • Refresh - Refreshes the data on the screen to the current values from the switch.
