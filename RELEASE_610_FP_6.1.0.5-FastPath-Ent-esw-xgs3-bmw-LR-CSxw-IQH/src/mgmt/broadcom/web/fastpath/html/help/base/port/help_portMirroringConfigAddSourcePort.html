Help for Multiple Port Mirroring - Add Source Ports

Configurable Data

    • Session - Specifies the selected Session ID for which source ports are to be added.
    • Source Port(s) - Specifies the configured port(s) with directions as mirrored port(s). Traffic of the source port(s) is sent to the probe port.
    • Source Port(s) - Specifies the configured port(s) as mirrored port(s). Traffic of the source port(s) is sent to the probe port.
      Up to source ports can be selected per session.
    • Direction - Specifies the direction of traffic on source port(s) which will be sent to the probe port. Possible values are :
      • Tx and Rx - Both Ingress and Egress traffic.
      • Rx - Ingress traffic only.
      • Tx - Egress traffic only.

Command Buttons

    • Add - Adds the selected source ports to the list of configured source ports.
    • Cancel - Takes no action and return to the previous screen.
