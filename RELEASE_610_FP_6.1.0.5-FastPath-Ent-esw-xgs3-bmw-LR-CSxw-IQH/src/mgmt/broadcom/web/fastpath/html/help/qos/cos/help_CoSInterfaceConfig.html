Help for Class of Service Interface Configuration

Selection Criteria

    • emweb:/USMWEB_UNIT_SLOT_PORT_DISPLAY; - Specifies all CoS configurable interfaces. The option "Global" represents the most recent global configuration settings. These may be overridden on a per-interface basis.

Configurable Data

    • Interface Shaping Rate - Specifies the maximum bandwidth allowed, typically used to shape the outbound transmission rate. This value is controlled independently of any per-queue maximum bandwidth configuration. It is effectively a second-level shaping mechanism. Default value is . Valid Range is in increments of . The value 0 means maximum is unlimited.
    • Queue Management Type - Specifies Queue depth management technique used for all queues on this interface. This is only used when device does not support the per-queue parameter.
      Queue Management Type can only be one of the following:
      • taildrop
      • wred
       Default value is .
    • WRED Decay Exponent - Specifies the decay exponent value used with the WRED average queue length calculation algorithm. Default value is . Valid Range is .

Command Buttons

    • Restore Defaults - Restores default settings.
    • Submit - Send the updated configuration to the switch. Configuration changes take effect immediately. These changes will not be retained across a power cycle unless a save is performed.
