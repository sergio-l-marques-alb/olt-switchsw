Help for MLD Snooping Multicast Router VLAN Status

Selection Criteria

    • Interface - The select box lists all Unit/Slot/Port Slot/Port .Select the interface for which you want to display the status.

Non-Configurable Data

    • VLAN ID - All VLAN IDs for which the Multicast Router Mode is Enabled
    • Multicast Router - Multicast Router Mode for VLAN ID.

Command Buttons

    • Refresh - Fetches the database and displays it again starting with the first entry in the table.
