Help for IGMP Snooping VLAN Configuration

Selection Criteria

    • VLAN ID - Specifies list of VLAN IDs for which IGMP Snooping is enabled.
    • Admin Mode - Enable IGMP Snooping for the specified VLAN ID. The Admin Mode is enabled by default.
    • Fast Leave Admin Mode - Enable or Disable the IGMP Snooping Fast Leave Mode for the specified VLAN ID. Fast Leave Admin Mode is disabled by default.

Configurable Data

    • VLAN ID - Appears when "New Entry" is selected in VLAN ID combo box. Specifies VLAN ID for which pre-configurable Snooping parameters are to be set.
    • Maximum Response Time - Sets the value for maximum response time of IGMP Snooping for the specified VLAN ID. Valid range is seconds. Its value should be less than Group Membership Interval value. Default value is seconds.
    • Group Membership Interval - Sets the value for group membership interval of IGMP Snooping for the specified VLAN ID. Valid range is seconds. The default value is seconds. The configured value must be greater than the Max Response Time.
    • Multicast Router Expiry Time - Sets the value for multicast router expiry time of IGMP Snooping for the specified VLAN ID. Valid range is seconds. Default value is .

Command Buttons

    • Submit - Updates the switch with the values you entered.
    • Delete - Updates the switch with the default values.
