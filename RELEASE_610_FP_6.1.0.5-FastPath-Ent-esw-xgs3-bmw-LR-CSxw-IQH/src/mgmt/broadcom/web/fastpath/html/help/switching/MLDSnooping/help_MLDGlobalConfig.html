MLD Help

Help for MLD Global Configuration

Configurable Data

    • Admin Mode - Select enable or disable from the pulldown menu to set the administrative status of MLD in the router to active or inactive. The default is disable.

Command Buttons

    • Submit - Sends the updated configuration to the router. Configuration changes take effect immediately. These changes will not be retained across a power cycle unless a save is performed.
    • Refresh - Refreshes the data on the screen with present state of data in the switch.
