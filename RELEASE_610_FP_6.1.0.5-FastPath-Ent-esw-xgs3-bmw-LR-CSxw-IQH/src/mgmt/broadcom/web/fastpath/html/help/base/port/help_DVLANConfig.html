Help for DVLAN Configuration

Selection Criteria

    • Global EtherType - This specifies the global EtherType available. The two-byte hex EtherType to be used as the first 16 bits of the DVLAN tag.
      • 802.1Q Tag - Commonly used tag representing 0x8100
      • vMAN Tag - Commonly used tag representing 0x88A8
      • Custom Tag - Configure the tag for EtherType by providing a custom value.

Configurable Data

    • Configure Default TPID - This will configure the selected Global EtherType as the Default TPID.
    • Custom Value - This will configure the custom tag in any range from .

Command Buttons

    • Submit - Updates the switch with the values on this screen. If you want the switch to retain the new values across a power cycle, you must perform a save.
    • Remove - Deletes the selected Global Ether Type.
    • Refresh - Refreshes the data on the screen with the present state of the data in the switch.
