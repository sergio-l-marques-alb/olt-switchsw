iSCSI Help

Help for iSCSI Add Targets

Configurable Data

    • TCP Port - TCP port number on which a iSCSI target listens to requests. TCP Port should be a whole number in the range 1 to 65535. Upto 16 TCP ports can be defined in the system. iSCSI well-known ports 3260 and 860 are configured as default but can be removed as any other configured target.
    • IP Address - IP address of the iSCSI target. IP address should in the format X.X.X.X only.
    • Target Name - iSCSI name of the iSCSI target. A name can be of maximum of 222 chararcters.

Command Buttons

    • Submit - Send the updated configuration to the switch. Configuration changes take effect immediately. These changes will not be retained across a power cycle unless a save is performed.
    • Back - Returns to the iSCSI Targets Table page.
