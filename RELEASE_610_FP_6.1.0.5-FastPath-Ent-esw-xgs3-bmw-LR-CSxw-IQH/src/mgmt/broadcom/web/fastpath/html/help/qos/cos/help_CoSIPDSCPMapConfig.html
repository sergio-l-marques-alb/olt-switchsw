Help for Class of Service IP DSCP Mapping Configuration

Selection Criteria

    • Unit/Slot/Port Slot/Port - Specifies all CoS configurable interfaces. The option "Global" represents the most recent global configuration settings. These may be overridden on a per-interface basis.

Configurable Data

    • Traffic Class - Specify which internal traffic class to map the corresponding IP DSCP value. Valid Range is .

Command Buttons

    • Submit - Send the updated configuration to the switch. Configuration changes take effect immediately. These changes will not be retained across a power cycle unless a save is performed.
    • Restore Defaults - Restores default settings.
