Help for System Resources


Use this panel to display the Memory Usage and Task CPU Utilization.

Non-Configurable Data

    • Free Memory - Available Free Memory on system.
    • Alloc Memory - Allocated Memory for the system.
    • Task Id - Id of the Currently Running Tasks.
    • Task Name - Name of the Currently Running Tasks.
    • CPU Utilization(%) - CPU Utilization of task in terms of Percentage.
    • Total CPU Utilization - Total CPU Utilization in terms of Percentage.

Command Buttons

    • Refresh - Refreshes the page with the latest data.
