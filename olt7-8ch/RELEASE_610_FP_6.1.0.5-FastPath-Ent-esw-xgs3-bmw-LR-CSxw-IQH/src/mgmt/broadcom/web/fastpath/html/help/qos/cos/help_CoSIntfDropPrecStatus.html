Help for Class of Service Interface Queue Drop Precedence Status

Selection Criteria

    • emweb:/USMWEB_UNIT_SLOT_PORT_DISPLAY; - Specifies all CoS configurable interfaces. The option "Global" represents the most recent global configuration settings. These may be overridden on a per-interface basis.
    • Queue ID - Specifies all the available queues.

Non-Configurable Data

    • Drop Precedence Level - Specifies the drop precedence level.
    • Tail Drop Threshold - Specifies the tail drop queue threshold above which all packets are dropped for the specified precedence level. Values are in sixteenths of overall device queue size.
    • WRED Minimum Threshold - Specifies the weighted RED minimum queue threshold below which no packets are dropped for the current drop precedence level. Values are in sixteenths of overall device queue size.
    • WRED Maximum Threshold - Specifies the weighted RED maximum queue threshold above which all packets are dropped for the current drop precedence level. Values are in sixteenths of overall device queue size.
    • WRED Drop Probability - Determines the packet drop probability for the current drop precedence level.
