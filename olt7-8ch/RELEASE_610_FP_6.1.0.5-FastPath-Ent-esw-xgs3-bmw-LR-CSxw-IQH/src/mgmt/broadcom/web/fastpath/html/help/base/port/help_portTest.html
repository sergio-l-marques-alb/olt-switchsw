Help for Loopback Test

Selection Criteria

    • Unit/Slot/Port Slot/Port - This field indicates the interface to which the Loopback(MAC/PHY) to be tested.

Configurable Data

    • Interface - Lists the interfaces to be tested in the Unit/Slot/Port Slot/Port notation.
    • Loopback Test Type - This Lists the Loopback Test type to be tested on the Port. User can select PHY-Loop to test Physical Layer level Loopback test on the port. User can select MAC-Loop to test MAC Layer level Loopback test on the port. User can also select No-Loop to avoid port is Looped.

Command Buttons

    • Run - Perform a Loopback test on the selected interface. The Loopback test may take up to 5 seconds to complete. If the port has an active link then the link is taken down. The command returns a statistics information like Data TX, RX, Errored TX, RX etc. Note that if the link is down still user can able to do Loopback test on the port.
