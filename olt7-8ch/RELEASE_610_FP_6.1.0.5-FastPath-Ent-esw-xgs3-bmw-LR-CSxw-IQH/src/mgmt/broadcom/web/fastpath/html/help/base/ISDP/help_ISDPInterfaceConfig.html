ISDP Help

Help for ISDP Interface Configuration

Configurable Data

Non-Configurable Data
    • Slot/Port - Specifies the list of ports on which ISDP can be configured.
    • ISDP Mode - An indication of whether the Industry Standard Discovery Protocol is currently ON/OFF on Interface.

Command Buttons

    • Submit - Applies the new configuration and causes the changes to take effect. These changes will not be retained across a power cycle unless a save configuration is performed.
    • Refresh - Updates the information on the page.
