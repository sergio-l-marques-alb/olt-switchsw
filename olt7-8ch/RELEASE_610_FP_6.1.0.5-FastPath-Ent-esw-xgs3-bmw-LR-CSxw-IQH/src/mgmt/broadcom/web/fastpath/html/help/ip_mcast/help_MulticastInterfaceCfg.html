Help For IPv4 Multicast Interface Configuration

Selection Criteria

    • emweb:/USMWEB_UNIT_SLOT_PORT_DISPLAY; - Select the routing interface you want to configure from the dropdown menu.
    • emweb:/USMWEB_UNIT_SLOT_PORT_DISPLAY; - Select the routing interface from the dropdown menu for which the configured TTL threshold need to be viewed.

Configurable Data

    • TTL Threshold - Enter the TTL threshold below which a multicast data packet will not be forwarded from the selected interface. You should enter a number between 0 and 255. If you enter 0 all multicast packets for the selected interface will be forwarded.
    • TTL Threshold - The TTL threshold below which a multicast data packet will not be forwarded from the selected interface.

Command Buttons

    • Submit - Sends the updated configuration to the router. Configuration changes take effect immediately. These changes will not be retained across a power cycle unless a save is performed.
    • Refresh - Refresh the data on the screen with the present state of the data in the router.
