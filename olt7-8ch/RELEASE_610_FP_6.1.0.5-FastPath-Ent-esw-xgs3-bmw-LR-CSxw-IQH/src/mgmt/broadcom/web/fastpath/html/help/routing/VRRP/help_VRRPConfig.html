VRRP Help

Help for VRRP Virtual Router Global Configuration

Configurable Data

    • Admin Mode - This sets the administrative status of VRRP in the router to active or inactive. Select enable or disable from the pull down menu. The default is disable.

Command Buttons

    • Submit - Send the updated configuration to the switch. Configuration changes take effect immediately. These changes will not be retained across a power cycle unless a save is performed.
    • Refresh - Refreshes the screen with updated configuration.
