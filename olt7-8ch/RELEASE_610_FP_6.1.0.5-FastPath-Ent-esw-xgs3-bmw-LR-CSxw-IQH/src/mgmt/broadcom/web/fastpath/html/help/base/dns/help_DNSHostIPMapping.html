HostName IP Mapping Summary Help

Help for DNS HostName IP Mapping Summary

Non-Configurable Data

      DNS Static Entries

    • Host Name - The Host Name of the static entry.
    • Inet Address - The IPv4 or IPv6 Address of the static entry.

Configurable Data

    • Remove Static - Removes the selected Host Name IP Mapping entry from the Host Name IP Mapping list.

Command Buttons

    • Add Static Entry - Loads Host Name IP Mapping configuration page to configure Host Name IP mapping entries.

Non-Configurable Data

      DNS Dynamic Entries

    • Host Name - The Host Name of the dynamic entry.
    • Total - Total time of the dynamic entry.
    • Elapsed - Elapsed time of the dynamic entry.
    • Type - The type of the dynamic entry.
    • Addresses - The Ipv4 or Ipv6 Address of the dynamic entries.

Configurable Data

    • Remove Dynamic - Removes the selected Host Name IP Mapping entry from the Host Name IP Mapping list.

Command Buttons

    • Submit - Update the switch with the values on the screen. If you want the switch to retain the new values across a power cycle you must perform a save.
    • Clear Dynamic Entries - This button removes all dynamic Host Name IP Mapping entries.
    • Refresh - Refresh the data on the screen to the current values from the switch.
