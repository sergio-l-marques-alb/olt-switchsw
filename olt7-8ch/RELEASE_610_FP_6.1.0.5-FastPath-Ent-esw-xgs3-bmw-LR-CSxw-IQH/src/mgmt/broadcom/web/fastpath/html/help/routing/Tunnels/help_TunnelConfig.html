Tunnel Configuration Help

Help for Tunnel Configuration

Tunnels can be created, configured and deleted from this page.

Selection Criteria

    • Tunnel - Select list of currently configured tunnel interfaces. 'Create' is also a valid choice if the maximum number of tunnel interfaces has not been created.
    • Mode - Selector for the Tunnel mode. The valid options are
      • Undefined - Undefined tunnel type(Tunnel is not operational).
      • 6-in-4-configured - Configured tunnel.
      • 6-to-4 - 6to4 automatic tunnel.
      By default it is 'Undefined'.
    • IPv6 Address - Select list of configured IPv6 addresses for the selected Tunnel interface. Add is also a valid choice if the maximum number of addresses has not been configured.
    • Source Type - Select the desired source type, Address or Interface. If Address is selected the source address for this tunnel must be entered in dotted decimal notation. If Interface is selected the source interface for this tunnel must be selected. The address associated with the selected interface will be used as the source address.

Configurable Data

    • Tunnel ID - When 'Create' is chosen from the tunnel selector this list of available tunnel ID's becomes visible.
    • IPv6 Mode - Enable IPv6 on this interface using the IPv6 address. This option is only configurable prior to specifying an explicit IPv6 address.
    • IPv6 Unreachables - Enable or Disable generation of ICMPv6 Unreachables on the tunnel.The default value is .
    • IPv6 Address - When 'Add' is chosen from the IPv6 Address selector this IPv6 address input field becomes visible. Address must be entered in the format of ipv6 prefix.
    • IPv6 Prefix Length - When 'Add' is chosen from the IPv6 Address selector this IPv6 Prefix Length input field becomes visible. This specifies the prefix length for the prefix entered in IPv6 Address field.
    • EUI64 - The user also has the option to specify the 64-bit extended unique identifier (EUI-64).
    • Source Address - The source address for this tunnel in dotted decimal notation.
    • Destination Address - The destination address for this tunnel in dotted decimal notation.

Command Buttons

  • Create Tunnel - Creates a tunnel interface.
  • Submit - Updates the system with the values on this screen.
  • Delete Tunnel - Removes the selected interface.
  • Add IPv6 Address - Configures IPv6 Address.
  • Delete Selected Address - Removes the selected IPv6 Address.
