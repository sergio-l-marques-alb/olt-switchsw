◆ bcm_mcast_addr_add()

int bcm_mcast_addr_add ( int  unit,
bcm_mcast_addr_t *  mcaddr 
)

Add/Remove an entry in the multicast table.

Add/remove an entry in the multicast table. Adds new ports in the specified entry to all routers Multicast address. This is available only on StrataXGS switch devices and it does not check that the MAC address is a valid multicast address.

Parameters
[in,out]unitUnit number.
[in,out]mcaddrData structure containing information to be written to the multicast entry.
Return values
BCM_E_NONEOperation completed successfully
BCM_E_INITModule is not initialized.
BCM_E_XXXOther errors
BCM_E_UNAVAILOn non StrataXGS switch devices

Set/Get/Delete the L2 multicast forwarding port bitmap.

Set the L2 multicast forwarding port bitmap for a given L2 multicast index and ingress port. Get the current L2 multicast forwarding port bitmap for a given L2 multicast index and ingress port. Delete a set of ports from the L2 multicast forwarding port bitmap for a given L2 multicast index and ingress port.

Parameters
[in,out]unitUnit number.
[in,out]mc_idxThe Multicast index carried in HiGig header
[in,out]in_portIngress port number
[in,out]pbmp(for "_set") IPMC forwarding port bitmap
(for "_get", OUT) IPMC forwarding port bitmap
Return values
BCM_E_NONEOperation completed successfully
BCM_E_UNAVAILOn non StrataXGS switch devices
