Create a packet format entry.

Create a packet format entry in the software. The entry will be installed only when the packet format entry is added to the udf. Please note that member <tunnel> of structure <pkt_format> must set a correlative of type value. Type value refers to the table BCM_PKT_FORMAT_TUNNEL_XXX. Note that If the <tunnel> in structure <pkt_format> type is BCM_PKT_FORMAT_TUNNEL_ANY, <inter_ip> in structure <pkt_format> must be BCM_PKT_FORMAT_IP_ANY. Note that If the <tunnel> in structure <pkt_format> type is BCM_PKT_FORMAT_TUNNEL_ANY, <outer_ip> in structure <pkt_format> must be BCM_PKT_FORMAT_IP_ANY. Note that If the <tunnel> in structure <pkt_format> type is BCM_PKT_FORMAT_TUNNEL_IP_IN_IP or BCM_PKT_FORMAT_TUNNEL_GRE, <outer_ip> and <inter_ip> in structure <pkt_format> must be BCM_PKT_FORMAT_IP4 or BCM_PKT_FORMAT_IP6. Note that If the <prio> in structure <pkt_format> is non-zero, entry with bigger <prio> has higher priority. If the <prio> in structure <pkt_format> is zero, entry with bigger <pkt_format_id> has higher priority.
