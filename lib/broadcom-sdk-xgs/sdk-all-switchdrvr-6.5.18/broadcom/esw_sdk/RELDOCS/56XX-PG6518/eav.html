Ethernet AV

The Ethernet AV Draft Standard 802.1as specifies protocols and procedures that ensure the timing requirements for audio and video applications on bridged networks.

The BCM EAV API provides a mechanism to configure devices which support this standard, including the capability to detect time synchronization control frames, set, and retrieve synchronization time stamps from data frames.

API Brief
bcm_eav_init(int unit) Initialize the Ethernet AV module and enable the EAV support globally.
bcm_eav_port_enable_get(int unit, bcm_port_t port, int* enable) Enable or disable EAV processing on a port
bcm_eav_port_enable_set(int unit, bcm_port_t port, int enable) Enable or disable EAV processing on a port
bcm_eav_link_status_get(int unit, bcm_port_t port, int* link) Get the EAV link status of the specified port.
bcm_eav_link_status_set(int unit, bcm_port_t port, int link) Set the EAV link status of the specified port
bcm_eav_control_get(int unit, bcm_eav_control_t type, uint32* arg, uint32* arg2) Configure EAV operating modes.
bcm_eav_control_set(int unit, bcm_eav_control_t type, uint32 arg, uint32 arg2) Configure EAV operating modes.
bcm_eav_timestamp_get(int unit, bcm_port_t port, uint32* timestamp) Get the timestamp value of the last egressed packet on the specified port.
bcm_eav_timesync_mac_get(int unit, bcm_mac_t eav_mac) Manage the MAC address for the Time Synchronization protocol. Get the destination MAC address for the Time Synchronization protocol.
bcm_eav_timesync_mac_set(int unit, bcm_mac_t eav_mac) Manage the MAC address for the Time Synchronization protocol. Get the destination MAC address for the Time Synchronization protocol.
bcm_eav_srp_mac_ethertype_get(int unit, bcm_mac_t mac, bcm_port_ethertype_t* ethertype) Manage the MAC address and EtherType used to trap SRP protocol packets.
bcm_eav_srp_mac_ethertype_set(int unit, bcm_mac_t mac, bcm_port_ethertype_t ethertype) Manage the MAC address and EtherType used to trap SRP protocol packets.
bcm_eav_pcp_mapping_get(int unit, bcm_eav_stream_class_t type, int* pcp, int* remapped_pcp) Manage the PCP(priority) value mapping for each EAV class packets.
bcm_eav_pcp_mapping_set(int unit, bcm_eav_stream_class_t type, int pcp, int remapped_pcp) Manage the PCP(priority) value mapping for each EAV class packets.
bcm_eav_bandwidth_get(int unit, bcm_port_t port, bcm_eav_stream_class_t type, uint32* bytes_sec, uint32* bytes_burst) Manage the reserved bandwidth for Class A or B stream traffic
bcm_eav_bandwidth_set(int unit, bcm_port_t port, bcm_eav_stream_class_t type, uint32 bytes_sec, uint32 bytes_burst) Manage the reserved bandwidth for Class A or B stream traffic
