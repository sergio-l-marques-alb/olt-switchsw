Detailed Description

L3 Key Structure.

Defines L3 table hash key structure.

  • For unicast addresses, key is DIP address or SIP to update HIT bit
  • For multicast addresses, the key is either IPMC group address (IPMC group address + src IP address), or (IPMC group address + src IP address + VID) depending on the chip being used and the (G, S, V) mode enable.

The BCM_L3_IP6 flag in l3k_flags is set to specify whether the IPv4 or IPv6 addresses are valid.
