Enable/disable the ability of packets to be sent out a port.

The bcm_auth_egress_set call should only be used around calls that a CPU uses to transmit an EAPOL frame out the port. If the port is in an unauthorized state with BCM_AUTH_BLOCK_IN set, or is not in an unauthorized state, the bcm_auth_egress_set call does nothing.

Parameters
[in,out]unitUnit number.
[in,out]portDevice port or logical port number
[in,out]enableTRUE for enable, FALSE for disable

int bcm_auth_mode_get ( int  unit,
int  port,
uint32 *  modep

Sets or gets authorization operating mode.

The available operating modes are described in the IEEE 802.1x specification. See the notes at the beginning of this section for more information on the treatment of the mode parameter. A port value of -1 or gport type for system configuration will configure all ports.

BCM authorization modes
Flags Additional info
BCM_AUTH_MODE_UNCONTROLLED
BCM_AUTH_MODE_UNAUTH the following additional flags may be logically ORed into the mode parameter:
  • BCM_AUTH_BLOCK_IN - Allow outgoing packets
  • BCM_AUTH_BLOCK_INOUT - Do not allow in or out packets (default)
BCM_AUTH_MODE_AUTH the following additional flags may be logically ORed into the mode parameter:
  • BCM_AUTH_LEARN
  • BCM_AUTH_IGNORE_LINK
  • BCM_AUTH_IGNORE_VIOLATION
  • BCM_AUTH_DROP_UNKNOWN
Parameters
[in,out]unitUnit number.
[in,out]portDevice port or logical port number, or -1 to set all non-stack ports
[in,out]modepOne of BCM_AUTH_MODE_XXX and other flags.

◆ bcm_auth_unauth_callback()

int bcm_auth_unauth_callback ( int  unit,
bcm_auth_cb_t  func,
void *  cookie 
)

Register a callback function to be called when a port becomes asynchronously unauthorized.

Registers a callback function to be called whenever an unauthorized MAC address is received. The cookie is passed to the callback routine along with one of the following reason codes:

  • BCM_AUTH_REASON_UNKNOWN
  • BCM_AUTH_REASON_LINK
  • BCM_AUTH_REASON_VIOLATION
Parameters
[in,out]unitUnit number.
[in,out]funcCallback function to register
[in,out]cookieArbitrary value passed to callback function
