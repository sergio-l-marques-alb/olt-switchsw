Modified Flex Counter Structures:

API Description
bcm_flexctr_action_t

Flex counter action structure.

This data structure lists the properties of a flex counter action.

bcm_flexctr_index_operation_t

Flex counter index operation structure.

This data structure lists the operation of a flex counter index. Flex counter index would be generated as below: value0 = (SEL(object0) >> shift0) & ((1 << mask_size0) - 1)). value1 = (SEL(object1) & ((1 << mask_size1) - 1)) << shift1. index = (value1 | value0).

bcm_flexctr_source_t

Enumeration of flex counter sources.

Flex counter source indicates the sources to generates a flex counter action.

bcm_flexctr_value_operation_type_t Enumeration of flex counter value operation types.
bcm_flexctr_object_t

Enumeration of flex counter objects.

Flex counter object could be used to generate counter index and counter value. Some objects could also be used for conditional trigger and quantization.

bcm_flexctr_quantization_t

Flex counter quantization structure.

This data structure lists the properties of a flex counter quantization. Each quantization instance generates a range check result.

bcm_flexctr_trigger_t

Flex counter trigger structure.

This data structure lists the properties of the flex counter trigger.

bcm_flexctr_value_operation_t

Flex counter value operation structure.

This data structure lists the operation of a flex counter update value. Flex counter update value would be generated as below: value0 = (SEL(object0) >> shift0) & ((1 << mask_size0) - 1)). value1 = (SEL(object1) & ((1 << mask_size1) - 1)) << shift1. value = (value1 | value0). New_counter = UPDATE_TYPE(old_counter, value).

bcm_flexctr_group_action_t

Flex counter group action structure.

This data structure lists the properties of a flex counter group action.

Flex state update value would be generated as below: value0 = (SEL(object0) >> shift0) & ((1 << mask_size0) - 1)). value1 = (SEL(object1) & ((1 << mask_size1) - 1)) << shift1. value = (value1 | value0). New_counter = UPDATE_TYPE(old_counter, value).

bcm_flexstate_object_t

Enumeration of flex state objects.

Flex state object could be used to generate state index and counter value. Some objects could also be used for conditional trigger and quantization.

bcm_flexstate_trigger_type_t Enumeration of flex state trigger type.
bcm_flexstate_counter_mode_t Enumeration of flex state counter modes.
bcm_flexstate_index_operation_t

Flex state index operation structure.

This data structure lists the operation of a flex state index. Flex state index would be generated as below: value0 = (SEL(object0) >> shift0) & ((1 << mask_size0) - 1)). value1 = (SEL(object1) & ((1 << mask_size1) - 1)) << shift1. index = (value1 | value0).

bcm_flexstate_value_operation_type_t Enumeration of flex state value operation types.
bcm_flexstate_group_action_t

Flex state group action structure.

This data structure lists the properties of a flex state group action.

bcm_flexstate_counter_value_t

Flex state statistic value.

Normal mode: value[0] is the data of counter value A. value[1] is the data of counter value B. Wide mode: value[0] is the upper part data of counter value A. value[1] is the lower part data of counter value A.
