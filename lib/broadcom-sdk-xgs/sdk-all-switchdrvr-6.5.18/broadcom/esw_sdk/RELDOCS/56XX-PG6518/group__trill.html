◆ bcm_trill_vpn_create()

int bcm_trill_vpn_create ( int  unit,
bcm_trill_vpn_config_t *  info 
)

Create a TRILL VPN.

Create a TRILL VPN. The bcm_trill_vpn_config_t structure is described below:

typedef struct bcm_trill_vpn_config_s {
uint32 flags; // Configuration flags
bcm_vpn_t vpn; // VPN ID.
bcm_vlan_t high_vid; // Outer vlan
bcm_vlan_t low_vid; // Inner vlan
bcm_multicast_t broadcast_group; // Broadcast group
bcm_multicast_t unknown_unicast_group; // Unknown unicast group
bcm_multicast_t unknown_multicast_group; // Unknown multicast group
bcm_policer_t policer_id; // Policer ID to be used
} bcm_trill_vpn_config_t;

A VPN is assigned a unique ID. which is returned in the vpn field. The multicast group(s) for broadcast and unknown unicast/multicast must first be created using the bcm_multicast_* APIs.

TRILL VPN Config Flags
Name Purpose
BCM_TRILL_VPN TRILL Fine Grained VPN.
BCM_TRILL_VPN_WITH_ID Create VPN with specified ID.
BCM_TRILL_VPN_REPLACE Used when modifying the attributes of an existing VPN.
BCM_TRILL_VPN_ACCESS Used for access side.For mapping vpn -> high+low vid
BCM_TRILL_VPN_NETWORK Used for network side."For mapping high+low vid -> vpn
BCM_TRILL_VPN_INVALID Invalid TRILL VPN .
BCM_TRILL_VPN_TRANSPARENT_SERVICE Create VPN with transparent service.
Parameters
[in,out]unitUnit number.
[in,out]info(IN/OUT) VPN info
