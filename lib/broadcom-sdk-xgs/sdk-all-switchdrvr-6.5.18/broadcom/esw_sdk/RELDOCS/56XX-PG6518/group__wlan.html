Add a wireless client to the database.

Add a wireless client to the database. bcm_wlan_client_t structure describes the wireless client attributes. The following configuration flags can be used:

Wireless Client Configuration Flags
Name Purpose
BCM_WLAN_CLIENT_REPLACE Must be set to update the attributes of an existing client.
BCM_WLAN_CLIENT_ROAMED_IN The client has roamed in, that is, this AC is the FA.
BCM_WLAN_CLIENT_ROAMED_OUT The client has roamed out, that is, this AC is the HA.
BCM_WLAN_CLIENT_AUTHORIZED The client is authorized.
BCM_WLAN_CLIENT_CAPTIVE The captive portal state.

◆ bcm_wlan_port_add()

int bcm_wlan_port_add ( int  unit,
bcm_wlan_port_t *  info 
)

Add a new WLAN port or update an existing one.

Add a new WLAN port or update an existing one. bcm_wlan_port_t structure describes the WLAN port or "virtual-port" attributes. The following configuration flags can be used:

WLAN Port Configuration Flags
Name Purpose
BCM_WLAN_PORT_WITH_ID Add / update a port with the specified ID.
BCM_WLAN_PORT_REPLACE Update the attributes of an existing port.
BCM_WLAN_PORT_DROP Drop all packets destined to this port.
BCM_WLAN_PORT_NETWORK Port is configured as a network port.
BCM_WLAN_PORT_BSSID For a WTP to AC port, match on the BSSID.
BCM_WLAN_PORT_BSSID_RADIO For a WTP to AC port, match on the BSSID and radio ID.
BCM_WLAN_PORT_MATCH_TUNNEL For an AC to AC port, match on the tunnel ID.
BCM_WLAN_PORT_EGRESS_CLIENT_MULTICAST Set the wireless client multicast bitmap (limit of 16) for AC to WTP ports.
BCM_WLAN_PORT_EGRESS_BSSID Include the optional CAPWAP radio MAC header in outgoing packets.
BCM_WLAN_PORT_EGRESS_TUNNEL_REMOTE Tunnel encapsulation is done remotely.
BCM_WLAN_PORT_ROAM_ENABLE Enable Roaming for the WLAN port
Parameters
[in,out]unitUnit number.
[in,out]info(IN/OUT) bcm_wlan_port_t
Return values
BCM_E_XXX
