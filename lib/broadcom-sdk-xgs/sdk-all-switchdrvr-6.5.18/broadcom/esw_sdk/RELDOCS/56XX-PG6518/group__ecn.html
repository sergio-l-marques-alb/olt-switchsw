int bcm_ecn_map_set ( int  unit,
uint32  options,
int  ecn_map_id,
bcm_ecn_map_t *  ecn_map 
)

To set an ECN mapping entry in an ECN mapping profile.

To set an ECN mapping entry in the specific profile. Ecn map configuration is listed in the following tables. BCM_ECN_MAP_ENTRY_f This ecn map structure can be used to set entries in four types of ecn mappings. MPLS EXP to IP ECN mapping (used in L2/L3 VPN LER termination and LSR ingress stage) ecn and exp value in bcm_ecn_map_t are used to get a new ecn. BCM_ECN_TRAFFIC_ACTION_INGRESS_DROP can be used in bcm_mpls_ecn_map_t.action_flags. IP ECN to EXP mapping (used in LER initiation) ecn and exp value in bcm_ecn_map_t are used to get a new exp. BCM_ECN_TRAFFIC_ACTION_EGRESS_DROP and BCM_ECN_TRAFFIC_ACTION_EGRESS_EXP_MARKING can be used in bcm_ecn_map_t.action_flags. INT CN to EXP mapping (used in LSR egress stage) int_cn and exp value in bcm_mpls_ecn_map_t are used to get a new exp. BCM_ECN_TRAFFIC_ACTION_EGRESS_DROP and BCM_ECN_TRAFFIC_ACTION_EGRESS_EXP_MARKING can be used in bcm_ecn_map_t.action_flags. Tunnel termination ECN mapping inner_ecn and ecn in bcm_mpls_ecn_map_t are used to get a new ecn. BCM_ECN_TRAFFIC_ACTION_INGRESS_ECN_MARKING and BCM_ECN_TRAFFIC_ACTION_INGRESS_DROP can be used in bcm_ecn_map_t.action_flags. For non-IP packets, ecn value in bcm_ecn_map_t and BCM_ECN_TRAFFIC_ACTION_EGRESS_DROP in bcm_ecn_map_t.nonip_action_flags are used to get an ECN DROP action.

ECN Map structure
Parameter Description
action_flags action flags for ECN.BCM_ECN_MAP_ACTION_FLAGS_f
int_cn internal cn as key.
inner_ecn inner packet ip ecn as key
ecn outer packet ip ecn as key
exp mpls_exp as key.
new_ecn new ecn value should be updated in ip packet.
new_exp new mpls_exp should be updated in MPLS packet.
nonip_action_flags action flags for ECN.BCM_ECN_MAP_ACTION_FLAGS_f for non-IP packets
Parameters
[in,out]unitUnit number.
[in,out]optionsBCM ecn options.
[in,out]ecn_map_idBCM ecn map id.
[in,out]ecn_map

◆ bcm_ecn_traffic_action_config_get()

int bcm_ecn_traffic_action_config_get ( int  unit,
bcm_ecn_traffic_action_config_t *  ecn_config 
)

Assign/Retrieve the actions to the specified ECN traffic.

Assign/Retrieve the actions to the specified ECN traffic. The ECN traffic is mapped by the packets ECN value and internal congestion notification(int_cn) value. Note that since this API can configure actions on various stages, it is possible that it is invoked multiple times, each time with parameters for a specific stage. Base on the action type, there are only several coresponding action flags that can be used to configure. For example, the BCM_ECN_TRAFFIC_ACTION_ENQUEUE_WRED_RESPONSIVE and BCM_ECN_TRAFFIC_ACTION_ENQUEUE_MARK_ELIGIBLE are used to be configured for action type, BCM_ECN_TRAFFIC_ACTION_TYPE_ENQUEUE. ECN traffic action configuration is listed in the following tables. BCM_ECN_TRAFFIC_ACTION_CONFIG_table

BCM ECN Traffic Action Configuration
Parameter Description
action_type action type. see table BCM_ECN_TRAFFIC_ACTION_TYPE_f
action_flags action flags of the action type. see table BCM_ECN_TRAFFIC_ACTION_FLAG_f
int_cn internal congestion notification(int_cn) value
color packet color
ecn ECN value of packet's IP header
new_ecn New ECN value for remarking. It is valid when BCM_ECN_TRAFFIC_ACTION_EGRESS_ECN_MARKING is set.
congested_int_cn New int_cn to be updated when the congestion is experienced. It is valid when BCM_ECN_TRAFFIC_ACTION_DEQUEUE_CONGESTION_INT_CN_UPDATE is set.
non_congested_int_cn New int_cn to be updated when the congestion is not experienced. It is valid when BCM_ECN_TRAFFIC_ACTION_DEQUEUE_NON_CONGESTION_INT_CN_UPDATE is set.

ECN Traffic Action Types
Type Description
BCM_ECN_TRAFFIC_ACTION_TYPE_ENQUEUE Action of enqueue stage.
BCM_ECN_TRAFFIC_ACTION_TYPE_DEQUEUE Action of dequeue stage.
BCM_ECN_TRAFFIC_ACTION_TYPE_EGRESS Action of egress stage.

ECN Traffic Action Flags
Flag Description
BCM_ECN_TRAFFIC_ACTION_NONE Action none.
BCM_ECN_TRAFFIC_ACTION_ENQUEUE_WRED_RESPONSIVE If set, enables WRED responsive dropping. Otherwise, enables WRED non-responsive dropping.
BCM_ECN_TRAFFIC_ACTION_ENQUEUE_MARK_ELIGIBLE If set, avoids WRED dropping when ECN marking is enabled.
BCM_ECN_TRAFFIC_ACTION_DEQUEUE_CONGESTION_INT_CN_UPDATE If set, updates the value of int_cn when congestion is experienced.
BCM_ECN_TRAFFIC_ACTION_DEQUEUE_NON_CONGESTION_INT_CN_UPDATE If set, updates the value of int_cn when congestion is not experienced.
BCM_ECN_TRAFFIC_ACTION_EGRESS_ECN_MARKING If set, indicates that the packet's ECN bits need to be updated.
BCM_ECN_TRAFFIC_ACTION_EGRESS_DROP If set, indicates to drop packet in egress stage.
BCM_ECN_TRAFFIC_ACTION_INGRESS_DROP If set, indicates to drop packet in ingress stage.
BCM_ECN_TRAFFIC_ACTION_EGRESS_EXP_MARKING If set, indicates that the packet's EXP bits need to be updated.
BCM_ECN_TRAFFIC_ACTION_INGRESS_ECN_MARKING If set, indicates that the packet's ECN bits need to be updated.
BCM_ECN_TRAFFIC_ACTION_RESPONSIVE If set, indicates that the configuration of this mapping is for ecn responsive traffic.
Parameters
[in,out]unitUnit number.
[in,out]ecn_config
