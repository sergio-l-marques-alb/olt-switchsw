◆ sal_strncat_s()

char* sal_strncat_s ( char *  dst,
const char *  src,
size_t  dest_max_size 
)

This function calculates the maximal number of characters that can be appended to 'dst' and, then, appends 'src' as per the calculated size. Also it always sets NULL ('0') as last character of the destination string ('dst'). If no character could be appended to 'dst' then return value is set to NULL.
