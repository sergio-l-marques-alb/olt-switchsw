Custom Callouts

The custom APIs are designed to provide an efficient means to batch collections of similar API operations when operating on remotely dispatched devices.

When executing APIs on remote devices, the API must be transmitted from the calling CPU to the remote CPU that actually connects to the target device. At least one packet must be sent in each direction for the API arguments and the return value to get to their respective destinations. Often a number of similar operations must be done on a single port or on all ports of a device.

The custom APIs allow at least 256 unsigned integers to be transmitted between the calling CPU and the called CPU. Each of these integers will be encoded and decoded appropriately at each CPU. After it is received at the called CPU, an application supplied callback function is called with the appropriate arguments. The direction of the integer arguments can be either to the called CPU (a _set operation) or from the called CPU (a _get operation).

An example of use of the custom API is to accelerate returning all SNMP statistics for a given port. The calling code would be:

int
custom_stat_get(int unit, bcm_port_t port,
int dosync, uint64 stats[snmpValCount], void *user_data)
{
int rv, argi, s;
uint32 args[snmpValCount*2];
rv = bcm_custom_port_get(int unit, bcm_port_t port, int type, int max_len, uint32* args, int* actual_len);
if (rv >= 0) {
argi = 0;
for (s = 0; s < snmpValCount; s++) {
COMPILER_64_SET(stats[s], args[argi], args[argi+1]);
argi += 2;
}
}
return rv;
}

Each BCM unit would need to have an appropriate custom handler installed using

rv = bcm_custom_register(int unit, bcm_custom_cb_t func, void* user_data);

The definition of custom_stat_handler would be:

int
custom_stat_handler(int unit, bcm_port_t port, int setget, int type, int length,
uint32 *args, int *returned_length, void *user_data)
{
int rv, argi, s;
uint64 val;
if (setget != BCM_CUSTOM_GET) {
return BCM_E_PARAM;
}
switch (type) {
case 2: /* sync stats counters, then get them */
rv = bcm_stat_sync(int unit);
if (rv < 0) {
break;
}
/* FALLTHROUGH */
case 1: /* just get all stats */
rv = 0;
argi = 0;
for (s = 0; s < snmpValCount; s++) {
rv = bcm_stat_get(int unit, bcm_port_t port, bcm_stat_val_t type, uint64* value);
if (rv < 0) {
break;
}
if (argi >= length-2) {
break;
}
args[argi++] = COMPILER_64_HI(val);
args[argi++] = COMPILER_64_LO(val);
}
break;
default:
rv = BCM_E_PARAM;
break;
}
return rv;
}
API Brief
bcm_custom_cb_t() Custom callback function prototype.
bcm_custom_register(int unit, bcm_custom_cb_t func, void* user_data) Register a custom callback function.
bcm_custom_unregister(int unit) Register a custom callback function.
bcm_custom_port_set(int unit, bcm_port_t port, int type, int len, uint32* args) Invoke a custom handler for a port
bcm_custom_port_get(int unit, bcm_port_t port, int type, int max_len, uint32* args, int* actual_len) Invoke a custom handler for a port
