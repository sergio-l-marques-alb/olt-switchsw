◆ credit_request_watchdog_delete_queue_thresh

int credit_request_watchdog_delete_queue_thresh

Watch Dog Delete Queue Threshold. When the Watch Dog Delete Queue Threshold time passed since the last credit for an active Queue, the Queue will be in Delete State. Range: 100-500. (0 disables TH). ◆ credit_request_watchdog_status_msg_gen

int credit_request_watchdog_status_msg_gen

Watch Dog Status Message Generation Period. When the Watch Dog Status Message Generation Period time passed since the last credit for an active Queue, the Queue will generate flow status message again. Range depends on mode. ARAD A supports 1-15 in aggressive status message credit watchdog mode, and 33-500 in the normal mode. (0 disables TH). Resolution: WD full cycle units.
