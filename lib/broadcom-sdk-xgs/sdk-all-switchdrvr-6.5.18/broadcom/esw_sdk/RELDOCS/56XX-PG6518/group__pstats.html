)

Create a Pstats session instance for SDK management.

Create a Pstats session with a set of session elements. All session elements will be recorded. Session data will be returned in the same order as the configurated session elements.

Parameters
[in,out]unitUnit number.
[in,out]optionsBCM_PSTATS_OPTIONS_REPLACE: replace existing. BCM_PSTATS_OPTIONS_WITH_ID: session_id argument is given.
[in,out]array_countNumber of session elements in array.
[in,out]element_arrayPSTATS session elements array.
[in,out]session_id(INOUT) id of a PSTATS session created via this API.
