◆ bcm_flow_encap_add()

int bcm_flow_encap_add ( int  unit,
bcm_flow_encap_config_t *  info,
uint32  num_of_fields,
bcm_flow_logical_field_t *  field 
)

Add a flow encap entry for L2 tunnel or L3 tunnel.

This API is to add the key based encapsulation data to the packet. The key generally consists of objects such as VFI,DVP,VRF,L3_OIF or objects specified by logical fields. One example is that the flow tunnel's VN_ID can be added to the packet based on the VFI object. The complete key material consists of flow_handle, flow_option, encap_criteria and key objects.

Parameters
[in,out]unitUnit number.
[in,out]info(IN/OUT) Flow encap configuration parameters
[in,out]num_of_fieldsnumber of elements of field array
[in,out]fieldfield array

◆ bcm_flow_encap_traverse()

int bcm_flow_encap_traverse ( int  unit,
bcm_flow_encap_traverse_cb  cb,
void *  user_data 
)
