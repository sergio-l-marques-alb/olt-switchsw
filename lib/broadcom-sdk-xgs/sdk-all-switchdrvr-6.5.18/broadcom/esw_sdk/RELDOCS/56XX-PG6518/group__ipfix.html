◆ bcm_ipfix_rate_create()

int bcm_ipfix_rate_create ( int  unit,
bcm_ipfix_rate_t *  rate_info 
)

Create an IPFIX flow rate meter entry.

Create an IPFIX flow rate meter entry. If both BCM_IPFIX_RATE_VIOLATION_REPLACE and BCM_IPFIX_RATE_VIOLATION_WITH_ID flags are specified, replace the entry with the ID provided in rate_id field of the bcm_ipfix_rate_t structure (rate_info argument). If neither of these 2 flags are set, a new allocated rate_id will be filled into the bcm_ipfix_rate_t structure on successful return.

Parameters
[in,out]unitUnit number.
[in,out]rate_info(IN/OUT) Pointer to rate information

int bcm_ipfix_rate_traverse ( int  unit,
bcm_ipfix_rate_traverse_cb  cb,
void *  userdata 
)

Traverse through IPFIX flow rate meter entries.

Traverse through IPFIX flow rate meter entries. The format of the bcm_ipfix_rate_traverse_cb is:

typedef int (*bcm_ipfix_rate_traverse_cb)(int unit,
bcm_ipfix_rate_t *rate_info,
void *userdata);

The bcm_ipfix_rate_t structure pointed by rate_info argument of callback function is no longer valid after the callback function exits. If application wants to save rate_info for later use, it must copy the content of the entire structure to application space.

Parameters
[in,out]unitUnit number.
[in,out]cbCallback function of type bcm_ipfix_rate_traverse_cb
[in,out]userdataArbitrary value passed to callback along with traversed data.

◆ bcm_ipfix_register()

int bcm_ipfix_register ( int  unit,
bcm_ipfix_callback_t  callback,
void *  userdata 
)

Register/Unregister a callback routine for BCM IPFIX export information.

Register/Unregister a callback routine that will be called whenever flow data is generated. The format of the bcm_ipfix_callback_t is:

typedef void (*bcm_ipfix_callback_t)(int unit,
bcm_ipfix_data_t *info,
void *userdata);

To unregister a callback, both callback and userdata must match the previously registered entry. Multiple callback routines can be registered by calling this API multiple times with different callback routines and/or different userdata.

Parameters
[in,out]unitUnit number.
[in,out]callbackCallback function of type bcm_ipfix_callback_t
[in,out]userdataArbitrary value passed to callback along with export data.
