Flex Digest

The flex digest serves the purpose of hashing based load balancing which splits micro-flows in a way that the traffic load is balanced across multiple output links. It takes packetâ€™s various fields and switch objects as input which is then fed into the hash function to generate a hash value.

API Brief
bcm_flexdigest_init(int unit) Initialize the flex digest module.
bcm_flexdigest_detach(int unit) Detach the flex-digest module.
