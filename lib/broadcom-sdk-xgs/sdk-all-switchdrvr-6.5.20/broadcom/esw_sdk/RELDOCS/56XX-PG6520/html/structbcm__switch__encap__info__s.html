Detailed Description

Encap info structure.

The following parameters are required when the encap type is MIML: egress_vlan, egress_srcmac, egress_dstmac, pkt_pri, pkt_cfi, pkt_type, iid, length and port.

The following parameters are required when the encap type is CUSTOM_HEADER: value32.

The other optional configurations depend on "flags" definition per service: e.g. qos_map_id , higig_vlan.
