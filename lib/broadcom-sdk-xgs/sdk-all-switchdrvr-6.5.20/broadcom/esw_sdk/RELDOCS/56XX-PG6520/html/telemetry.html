Telemetry APIs usage

Telemetry Overview

Telemetry is a module in SDK which will hold the details of Telemetry instances configured which are used to export telemetry stats to collector.

The SDK APIs described below achieve configuration of Telemetry instances and their association with collector export information.

Module initialization and detach

API Brief
bcm_telemetry_init(int unit) Initializes telemetry module.
bcm_telemetry_detach(int unit) Detach telemetry module.
bcm_telemetry_config_t_init(bcm_telemetry_config_t* telemetry_config) Initializes a telemetry config information structure.
bcm_telemetry_config_set(int unit, int telemetry_instance, int core, int config_count, bcm_telemetry_config_t* telemetry_config_list) Creates a telemetry instance.
bcm_telemetry_config_get(int unit, int telemetry_instance, int core, int max_count, bcm_telemetry_config_t* telemetry_config_list, int* config_count) Fetches a telemetry instance whose instance ID is provided.
bcm_telemetry_export_config_set(int unit, int telemetry_instance, bcm_collector_t collector_id, int export_profile_id) Configures a telemetry instance export information.
bcm_telemetry_export_config_get(int unit, int telemetry_instance, bcm_collector_t* collector_id, int* export_profile_id) Fetches a telemetry instance export information.
bcm_telemetry_export_config_delete(int unit, int telemetry_instance, bcm_collector_t collector_id, int export_profile_id) Deletes a telemetry instance export information.
bcm_telemetry_system_id_set(int unit, int system_id_len, uint8* system_id) Configures telemetry system ID information.
bcm_telemetry_system_id_get(int unit, int max_system_id_len, int* system_id_len, uint8* system_id) Fetches telemetry system ID information.
bcm_telemetry_instance_export_stats_get(int unit, int telemetry_instance, bcm_collector_t collector_id, bcm_telemetry_instance_export_stats_t* stats) Get the telemetry instance export statistics.
bcm_telemetry_instance_export_stats_set(int unit, int telemetry_instance, bcm_collector_t collector_id, bcm_telemetry_instance_export_stats_t* stats) Set the telemetry instance export statistics.
