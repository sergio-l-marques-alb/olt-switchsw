◆ bcm_failover_multi_level_attach()

int bcm_failover_multi_level_attach ( int  unit,
bcm_failover_multi_level_t  multi_level_failover 
)

Attaches multiple levels of 2 level protection switching.

This API binds two levels of multi level protection switching. User creates 2 child failover groups(with api bcm_failover_create) for second level of protection switching. Another multi level failover Id (with api bcm_failover_create and flag BCM_FAILOVER_MULTI_TYPE) is created as the first level of failover. First level can switch between two second level failover groups. Then both of child failover groups are passed in this API with multi level failover id to set parent/child links.

Parameters
[in]unitUnit number.
[in]multi_level_failoverPointer to bcm_failover_multi_level_t structure

Configure the failover ring on the system.

Setup the system failover ring with the matching vlans and ports by passing the appropriate flags. If the flag BCM_FAILOVER_VPN_TYPE is passed, we would not consider ports and vpn_vector information as we would perform forced flooding/learn disabling for all VPNs.
