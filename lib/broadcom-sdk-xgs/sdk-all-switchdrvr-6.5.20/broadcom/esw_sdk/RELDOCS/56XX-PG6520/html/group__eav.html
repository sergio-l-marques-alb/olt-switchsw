int bcm_eav_control_get ( int  unit,
bcm_eav_control_t  type,
uint32 *  arg,
uint32 *  arg2 
)

Configure EAV operating modes.

These APIs are used to get and set the configuration for a specific EAV control type. The supported EAV control types and arguments are defined as follows:

bcm_eav_control_t
EAV Control Type Description
bcmEAVControlMaxFrameSize Maximum supported frame size
bcmEAVControlTimeBase Timestamp for time synchronization protocol
bcmEAVControlTimeAdjust Adjust to the timestamp ticks
bcmEAVControlTickCounter Current timestamp ticks
bcmEAVControlSlotNumber Slot number in EAV
bcmEAVControlMacroSlotTime The slot time of a Macro slot for class 4 traffic
bcmEAVControlSlotAdjust Number of slots to apply the alerted slot adjustment
bcmEAVControlStreamClassAPCP ClassA PCP value for EAV
bcmEAVControlStreamClassBPCP ClassB PCP value for EAV

Arguments to the bcm_eav_control API
EAV Control Type arg1 arg2
bcmEAVControlMaxFrameSize The configured max frame size N/A
bcmEAVControlTimeBase 32-bit free running clock N/A
bcmEAVControlTimeAdjust Incremental time added to the Time Base in each running tick The number of ticks to apply the adjusted Time Increment
bcmEAVControlTickCounter The tick counter that defines when to increase the slot number N/A
bcmEAVControlSlotNumber The slot number for EAV N/A
bcmEAVControlMacroSlotTime The slot time of a Macro slot for Class 4 traffic. (0 for 1ms, 1 for 2ms and 2 for 4ms) N/A
bcmEAVControlSlotAdjust Define when the slot number counter will increase by 1 (0 for 3125, 1 for 3126 and 2 for 3124) The number of slots to apply the alerted slot adjustment
bcmEAVControlStreamClassAPCP The ClassA PCP value for EAV(0 ~ 7) N/A
bcmEAVControlStreamClassBPCP The ClassB PCP value for EAV(0 ~ 7) N/A

Arguments to the bcm_eav_control API for Hawkeye
EAV Control Type arg1 arg2
bcmEAVControlTimeAdjust Mode (0 for free running, 1 for phase adjustment, 2 and 3 for frequency adjustment) Operation attributes
Parameters
[in]unitUnit number.
[in]typeDevice port number.
[out]arg
[out]arg2
