In-band Flow Analyzer(IFA) Application

In-band Flow Analyzer Overview

In-band Flow Analyzer(IFA) is an embedded application that runs on embedded ARM core 0 of BCM56870 devices which either do ingress processing (by insert INT header) or egress process (sending INT packet to collector).

The SDK APIs described below will achieve configuration of INT header, Meta Data fields and reachability information of collector(s).

Module initialization and detach

API Brief
bcm_ifa_init(int unit) Initializes ifa subsystem.
bcm_ifa_detach(int unit) Detaches ifa subsystem.
bcm_ifa_header_t_init(bcm_ifa_header_t *header) Initialize the bcm_ifa_header_t struct.
bcm_ifa_header_create(int unit, uint32 options, int *header_id, bcm_ifa_header_t *header) Create the IFA header.
bcm_ifa_header_get(int unit, int header_id, bcm_ifa_header_t *header) Get the IFA header.
bcm_ifa_header_destroy(int unit, int header_id) Destroy the IFA header.
bcm_ifa_header_traverse(int unit, bcm_ifa_header_traverse_cb cb, void *user_data) Traverse all IFA headers.
