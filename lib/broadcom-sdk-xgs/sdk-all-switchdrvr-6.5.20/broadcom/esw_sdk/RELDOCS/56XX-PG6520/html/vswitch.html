Virtual Switching Instances

Some devices support virtual switching instances. Each virtual switching instance (VSI) operates as an independent switch. VSIs can be created and destroyed, and ports can be added to them using the APIs described in this section.

API Brief
bcm_vswitch_init(int unit) Initialize vswitch module.
bcm_vswitch_detach(int unit) Detach vswitch module.
bcm_vswitch_create(int unit, bcm_vlan_t* vsi) Create a Virtual Switching Instance.
bcm_vswitch_create_with_id(int unit, bcm_vlan_t vsi) Create a Virtual Switching Instance, with a specified ID.
bcm_vswitch_destroy(int unit, bcm_vlan_t vsi) Destroy a Virtual Switching Instance.
bcm_vswitch_destroy_all(int unit) Destroy all Virtual Switching Instances.
bcm_vswitch_port_traverse_cb() Vswitch callback function prototype
bcm_vswitch_port_traverse(int unit, bcm_vlan_t vsi, bcm_vswitch_port_traverse_cb cb, void* user_data) Traverse existing ports on vswitch
bcm_vswitch_port_add(int unit, bcm_vlan_t vsi, bcm_gport_t port) Add a logical port to the specified virtual switching instance.
bcm_vswitch_port_delete(int unit, bcm_vlan_t vsi, bcm_gport_t port) Remove a logical port from the specified virtual switching instance.
bcm_vswitch_port_delete_all(int unit, bcm_vlan_t vsi) Remove all logical port members from the specified virtual switching instance.
bcm_vswitch_port_get(int unit, bcm_gport_t port, bcm_vlan_t* vsi) Get the virtual switching instance of which the specified logical port is a member.
bcm_vswitch_cross_connect_add(int unit, bcm_vswitch_cross_connect_t* gports) Attach two given logical ports in a point-to-point service.
bcm_vswitch_cross_connect_get(int unit, bcm_vswitch_cross_connect_t* gports) Return peer, if protected return primary port, invalid gport is populated.
bcm_vswitch_cross_connect_delete(int unit, bcm_vswitch_cross_connect_t* gports) Remove attachment between the given logical ports.
bcm_vswitch_cross_connect_delete_all(int unit) Delete all point-to-point services.
bcm_vswitch_cross_connect_traverse(int unit, bcm_vswitch_cross_connect_traverse_cb cb, void* user_data) Traverse all existing point-to-point services.
bcm_vswitch_flexible_connect_add(int unit, uint32 flags, bcm_vswitch_flexible_connect_match_t* port_match, bcm_vswitch_flexible_connect_fwd_t* port_fwd) Attach given match port to forward destination.
bcm_vswitch_flexible_connect_get(int unit, uint32 flags, bcm_vswitch_flexible_connect_match_t* port_match, bcm_vswitch_flexible_connect_fwd_t* port_fwd) Return forwarding information for given match peer
bcm_vswitch_flexible_connect_delete(int unit, bcm_vswitch_flexible_connect_match_t* port_match) Remove flexible connection.
