◆ profile_id_mask

int profile_id_mask

This mask is used to calculate the EP Recirculate profile id for the drop events. Final profile id = ((ing_profile_id & profile_id_mask) | profile_id_offset). Where ing_profile_id is the original id derived from the ingress pipeline.

Definition at line 57592 of file switch.h.

◆ profile_id_offset

int profile_id_offset

This offset is used to calculate the EP Recirculate profile id for the drop events. See above for how the final profile id is calculated.

int threshold
