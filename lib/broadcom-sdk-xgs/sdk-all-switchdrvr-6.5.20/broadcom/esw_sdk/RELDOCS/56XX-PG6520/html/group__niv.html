Get NIV egress objects associated with a NIV port.

Retrieve NIV egress objects associated with a NIV port. If array_size = 0 and niv_egress_array = NULL, the number of NIV egress objects associated with the NIV port is returned in the parameter count. If array_size = -1 and with NIV port, port and virtual_interface_id input, a specific egress object can be retrieved, and the parameter count = 1.

Parameters
[in]unitUnit number.
[in]niv_portNIV GPORT ID
[in]array_sizeNumber of NIV egress objects allocated in niv_egress_array
[in,out]niv_egress_array(IN/OUT) Array of NIV egress objects
[out]countNumber of NIV egress objects returned

Create a set of NIV egress objects and add them to a NIV port.

Delete existing NIV egress objects associated with the NIV port. Then, create a set of new NIV egress objects and add them to the NIV port. In the niv_egress_array, the ID of each NIV egress object is returned in the egress_if field.

Parameters
[in]unitBCM device number
[in]niv_portNIV GPORT ID
[in]array_sizeNumber of NIV egress objects in niv_egress_array
[in,out]niv_egress_array(IN/OUT) Array of NIV egress objects
