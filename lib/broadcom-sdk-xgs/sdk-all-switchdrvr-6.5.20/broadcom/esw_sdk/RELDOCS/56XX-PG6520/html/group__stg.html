◆ bcm_stg_init()

int bcm_stg_init ( int  unit)

Initialize the Spanning Tree Group (STG) module to its initial configuration.

Initializes the STG module. A default STG is then created, with all its ports in the BCM_STG_STP_DISABLE state. Any previously existing STG information is lost.

Parameters
[in]unitUnit number.

◆ bcm_stg_list()

int bcm_stg_list ( int  unit,
bcm_stg_t **  list,
int *  count 
)

Generates a list of IDs for currently defined STGs.

Returns a (pointer to a) sorted list of currently active STGs. If there are no currently defined STGs, *list will be NULL and *count is 0. Otherwise, *list will point to an array of the *count currently defined STG IDs. The caller is responsible for freeing the memory allocated for the returned list, that is, calling bcm_stg_list_destroy .

Parameters
[in]unitUnit number.
[out]listPointer to returned pointer-to-array of STG IDs
[out]countPointer to returned count of STG IDs

Destroys a previously returned list of STG IDs; the STGs themselves are not affected.

Destroys a list of STG IDs previously returned by bcm_stg_list, effectively freeing the memory previously allocated by bcm_stg_list() for the list of STGs. The status of each STG in the list is not affected.

int bcm_stg_vlan_add ( int  unit,
bcm_stg_t  stg,

)

Add a VLAN to a specified STG.

Adds the specified VLAN to the specified STG. Both VLAN and STG must be currently defined. The VLAN is implicitly removed from the STG it currently belongs to before it is added to the new STG. Note: STG Id 0 is reserved, created during initialization, and used for internal purposes. VLANs should not be added to this group.

Parameters
[in]unitUnit number.
[in]stgSTG ID
[in]vidVLAN ID

Generates a list of VLANs in a specified STG.

Returns a pointer to a sorted list of VLANs belonging to the specified STG. If the STG currently contains no VLANs, *list is NULL and *count is 0. Otherwise, *list points to an array of the *count VLAN IDs in the indicated STG. The indicated STG must be currently defined. The caller is responsible for freeing the memory allocated for the returned list, that is, calling bcm_stg_vlan_list_destroy .

[out]listPointer to returned pointer-to-array of VLAN IDs
