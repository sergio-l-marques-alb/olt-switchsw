uint32 cw_s0_errs

Number of Code Words with S + 0 symbol errors where S is the selected value via symb_err_stats_sel in bcm_port_fdr_config_t. If symb_err_stats_sel is 0, S is 0. If symb_err_stats_sel is 1, S = 9 or S = 8 based on the chip.
