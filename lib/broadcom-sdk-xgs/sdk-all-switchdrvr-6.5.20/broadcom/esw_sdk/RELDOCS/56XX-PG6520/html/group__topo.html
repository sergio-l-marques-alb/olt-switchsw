int bcm_topo_map_set ( bcm_topo_map_f  _map)

Set or Get Topology mapping lookup function.

Set or get the topology mapping function that is appropriate for the local board. If NULL is used as the function to set, then a default topology mapping lookup function is used. The default function understands many common single board configurations using the various bcm_stk_xxx routines.

Parameters
[in]_map(for "_set") Mapping function matching the bcm_topo_map_f prototype
(for "_get", OUT) Mapping function
Return values
BCM_E_XXX

Lookup a path to a destination module.

Call the currently set topology mapping lookup function to find a port to use to reach a destination module number.

Parameters

[in]dest_modidFabric module ID to which the packet is to be routed
[out]exit_portThe physical port on unit on which the packet should exit to reach dest_modid
