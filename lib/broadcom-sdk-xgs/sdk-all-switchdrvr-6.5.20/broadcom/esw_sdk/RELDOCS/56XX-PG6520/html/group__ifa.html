◆ bcm_ifa_export_template_create()

int bcm_ifa_export_template_create ( int  unit,
uint32  options,
bcm_ifa_export_template_t *  id,
uint16  set_id,
int  num_export_elements,
bcm_ifa_export_element_info_t *  list_of_export_elements 
)

Create a ifa export template.

If BCM_IFA_EXPORT_TEMPLATE_WITH_ID is not set, an ID is created for the template. Otherwise, the specified ID is used. This API creates an export template with the list of information elements specified and marks it with the set_id in the IPFIX packets that are exported with this template.

Parameters
[in]unitBCM device number
[in]optionsoptions bitmap of BCM_IFA_EXPORT_TEMPLATE_XXX macros.
[in,out]id(IN/OUT) The ID of template created. IN parameter if options parameter has BCM_IFA_EXPORT_TEMPLATE_WITH_ID option set.
[in]set_idThe set_id of the template in the IPFIX packet.
[in]num_export_elementsThe number of export information elements present in the list_of_export_elements.
[in]list_of_export_elementsThe list of export information elements which forms the template.
