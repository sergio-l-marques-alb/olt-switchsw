Collector APIs usage

Collector Overview

Collector is a module in SDK which will hold the details of export and encapsulation details needed to reach it.

The SDK APIs described below achieve configuration of export profile and reachability information of collector(s).

Module initialization and detach

API Brief
bcm_collector_init(int unit) Initializes collector module.
bcm_collector_detach(int unit) Detach collector module.
bcm_collector_export_profile_create(int unit, uint32 options, int* export_profile_id, bcm_collector_export_profile_t* export_profile_info) Creates an export profile with given export information.
bcm_collector_export_profile_get(int unit, int export_profile_id, bcm_collector_export_profile_t* export_profile_info) Fetches an export profile information whose ID is provided.
bcm_collector_export_profile_ids_get_all(int unit, int max_size, int* export_profile_ids_list, int* list_size) Get all the configured collector export profiles.
bcm_collector_export_profile_destroy(int unit, int export_profile_id) Destroy a collector export profile.
bcm_collector_create(int unit, uint32 options, bcm_collector_t* collector_id, bcm_collector_info_t* collector_info) Create a collector.
bcm_collector_get(int unit, bcm_collector_t id, bcm_collector_info_t* collector_info) Gets a Collector information.
bcm_collector_get_all(int unit, int max_size, bcm_collector_t* collector_list, int* list_size) Get all the configured collectors.
bcm_collector_destroy(int unit, bcm_collector_t id) Destroys a collector.
bcm_collector_export_record_t_init(bcm_collector_export_record_t* record) bcm_collector_export_record_t_init
bcm_collector_callback_options_t_init(bcm_collector_callback_options_t* callback_options) bcm_collector_callback_options_t_init
bcm_collector_export_record_register(int unit, bcm_collector_t collector_id, bcm_collector_callback_options_t callback_options, bcm_collector_export_record_cb_f callback_fn, void* userdata) Register callback routine for Local collector record export.
bcm_collector_export_record_unregister(int unit, bcm_collector_t collector_id, bcm_collector_callback_options_t callback_options, bcm_collector_export_record_cb_f callback_fn) Unregister callback routine for Local collector record export.
