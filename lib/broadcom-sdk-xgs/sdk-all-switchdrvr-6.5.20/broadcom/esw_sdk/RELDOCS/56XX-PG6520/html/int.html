INT Application

INT Turnaround Overview

INT Turnaround is an embedded application that runs on embedded ARM core 0 of BCM56980 devices which do probe turn-around.

The SDK APIs described below will achieve configuration of INT header.

INT Turnaround Overview

This feature is included in the default FEATURE_LIST for devices that support it.

INT APIs allow application developers to manage In-band Network Telemetry features, including In-band Flow Analyzer, In-site Operations, Administration, and Maintenance and In-band Network Telemetry Data-plane Probe.
