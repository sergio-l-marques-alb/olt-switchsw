Instrumentation Configuration

The port related APIs provide control over a significant number of port attributes.

Visibility Configuration

API calls are supplied to set and get different visibility options

bcm_vis_mode_t
Name Value Description
VisModeNone 0 No visibility enabled
VisModeAlways 1 Last packet from any source will be always recorded
VisModeSelective 2 Packet wil be recording according to specific port configuration
API Brief
bcm_instru_vis_mode_set() Set/Get visibility mode
bcm_instru_vis_mode_get() Set/Get visibility mode
instru_control_t
Name Value Description
InstruControlIptSwitchId 0 IPT switch ID
InstruControlVisMode 1 Visibility Mode
InstruControlVisSampling 2 Visibility sampling parameters in Selective mode
InstruControlSFlowSamplingRate 3 Global sampling rate for sFlow datagrams
instru_vis_mode_control_t
Name Value Description
InstruVisModeNone 0 No visibility enabled
InstruVisModeAlways 1 Visibility is enabled for any packet from any port
InstruVisModeSelective 2 Selective visibility enabled, user port control to enable specific resources
API Brief
bcm_instru_control_set(int unit, uint32 flags, bcm_instru_control_t type, int arg) Set/Get visibility mode
bcm_instru_control_get(int unit, uint32 flags, bcm_instru_control_t type, int* arg) Set/Get visibility mode
instru_gport_control_t
Name Value Description
InstruGportControlTraceProbability 0 Probability in one tenth percentage
InstruGportControlVisEnable 1 Enable visibility on per port basis
InstruGportControlVisForce 2 Force visibility on per port basisspecific resources
InstruGportControlIptTrapToRcyEnable 2 Trap IPT packet to Recycle interface
API Brief
bcm_instru_gport_control_set(int unit, bcm_gport_t gport, uint32 flags, bcm_instru_gport_control_t type, int arg) Set/Get visibility mode
bcm_instru_gport_control_get(int unit, bcm_gport_t gport, uint32 flags, bcm_instru_gport_control_t type, int* arg) Set/Get visibility mode
instru_ipt_node_type_t
Name Value Description
InstruIptNodeTypeTail 0 Tail
InstruIptNodeTypeIntOverVxlanFirst 1 Vxlan first node
InstruIptNodeTypeIntOverVxlanIntermediate 2 Vxlan intermediate node
InstruIptNodeTypeIntOverVxlanLast 3 Vxlan last node
