Rate Limiting

The BCM rate layer provides access to the rate limiting features of the Broadcom switches. Rate limiting is useful to contain the effects of packet flooding by a malicious or malfunctioning network interface card.

Rate limiting can be applied individually to destination lookup failure (DLF), multicast (MCAST), and broadcast (BCAST) packets.

The following is the rate control BCM API related data structure:

typedef struct bcm_rate_limit_s {
int br_dlfbc_rate;
int br_mcast_rate;
int br_bcast_rate;
int flags;
} bcm_rate_limit_t;

RateLimitFlags lists the flags for the rate-limiting functions.

Rate Control Flags
Name Purpose
BCM_RATE_DLF Destination lookup failure
BCM_RATE_MCAST Multicast
BCM_RATE_BCAST Broadcast
BCM_RATE_UCAST Unicast
BCM_RATE_SALF Source address lookup failure
BCM_RATE_RSVD_MCAST Reserved multicast
BCM_RATE_KNOWN_MCAST Known multicast
BCM_RATE_UNKNOWN_MCAST Unknown multicast
BCM_RATE_CTRL_BUCKET_1 Bucket #1 in the multi-bucket system
BCM_RATE_CTRL_BUCKET_2 Bucket #2 in the multi-bucket system
BCM_RATE_MODE_PACKETS Packet mode
BCM_RATE_COLOR_BLIND Rate limiting is color blind
BCM_RATE_PKT_ADJ_HEADER_TRUNCATE Indicates that this Eth. Policer needs to be considerate of header truncate adjustment
typedef struct bcm_rate_packet_s {
uint32 flags;
int pps;
int kbits_burst;
} bcm_rate_packet_t;

RatePacketFlags lists the flags for the rate-packet functions.

Rate Packet Control Flags
Name Purpose
BCM_RATE_DLF Destination lookup failure
BCM_RATE_MCAST Multicast
BCM_RATE_BCAST Broadcast
API Brief
bcm_rate_packet_t_init(bcm_rate_packet_t* pkt_rate) Initialize a bcm_rate_packet_t structure.
bcm_rate_packet_set(int unit, bcm_port_t port, bcm_rate_packet_t* pkt_rate) Front ends to bcm_rate_packet_set/get functions. Uses a single data structure to write into rate control registers
bcm_rate_packet_get(int unit, bcm_port_t port, bcm_rate_packet_t* pkt_rate) Front ends to bcm_rate_packet_set/get functions. Uses a single data structure to write into rate control registers
bcm_rate_set(int unit, int pps, int flags) Configure/retrieve rate limit and on/off state of DLF, MCAST, and BCAST limiting.
bcm_rate_get(int unit, int* pps, int* flags) Configure/retrieve rate limit and on/off state of DLF, MCAST, and BCAST limiting.
bcm_rate_limit_t_init(bcm_rate_limit_t* rate_limit) Initialize a bcm_rate_limit_t structure.
bcm_rate_type_set(int unit, bcm_rate_limit_t* rl) Front ends to bcm_*_rate_set/get functions. Uses a single data structure to write into all the 3 rate control registers
bcm_rate_type_get(int unit, bcm_rate_limit_t* rl) Front ends to bcm_*_rate_set/get functions. Uses a single data structure to write into all the 3 rate control registers
bcm_rate_mcast_set(int unit, int pps, int flags, int port) Configure or retrieve rate limit for specified packet type on given port.
bcm_rate_bcast_set(int unit, int pps, int flags, int port) Configure or retrieve rate limit for specified packet type on given port.
bcm_rate_dlfbc_set(int unit, int pps, int flags, int port) Configure or retrieve rate limit for specified packet type on given port.
bcm_rate_mcast_get(int unit, int* pps, int* flags, int port) Configure or retrieve rate limit for specified packet type on given port.
bcm_rate_dlfbc_get(int unit, int* pps, int* flags, int port) Configure or retrieve rate limit for specified packet type on given port.
bcm_rate_bcast_get(int unit, int* pps, int* flags, int port) Configure or retrieve rate limit for specified packet type on given port.
bcm_rate_bandwidth_set(int unit, bcm_port_t port, int flags, uint32 kbits_sec, uint32 kbits_burst) Configure/retrieve metering rate limit for specified packet type on given port.
bcm_rate_bandwidth_get(int unit, bcm_port_t port, int flags, uint32* kbits_sec, uint32* kbits_burst) Configure/retrieve metering rate limit for specified packet type on given port.
