◆ name

uint8 name[BCM_OAM_GROUP_NAME_LENGTH]

This is equivalent to the Maintenance Association ID (MAID) in 802.1ag. Note that this is not a null-terminated string but an array of bytes. To avoid memory corruption, do not use string copy to populate this field.

◆ rx_name

uint8 rx_name[BCM_OAM_GROUP_NAME_LENGTH]

Additional MAID used for received OAM packets. It is used when the flag BCM_OAM_GROUP_RX_NAME is set. When used, the received MAID values are compared to this MAID instead of the 'name' field.
