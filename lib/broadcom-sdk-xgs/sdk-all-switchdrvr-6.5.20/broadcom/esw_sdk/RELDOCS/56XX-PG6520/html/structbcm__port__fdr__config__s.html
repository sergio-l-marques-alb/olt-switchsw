◆ symb_err_stats_sel

int symb_err_stats_sel

To select the stats range of symbol errors in bcm_port_fdr_stats_t, in which cw_sN_errs indicates count of code words with S+N symbol errors. symb_err_stats_sel = 0 indicates S is 0. symb_err_stats_sel = 1 indicates S = 9 or S = 8 based on the chip. The cw_s8_errs is 0 when the chip-dependent value of S is 8.
