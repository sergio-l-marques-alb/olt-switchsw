MACSec API

Introduction to Broadcom MACSec API

The Broadcom MACSec API consists of set of APIs which can used to configure MACSec functionality for Broadcom devices that support MACSec feature. All supported API calls are documented in this manual. Functions not defined in this document are NOT API calls and subject to change or removal at any time in any release. Functions defined in this document are intended, as much as reasonably possible, to be carried forward and supported on all platforms.

All MACSec API identifiers have prefix bcm_macsec_.

The API exports silicon features in a device independent manner as much as possible, but does not implement in software those functions that the underlying devices do not support.

MACSec APIs can be classified in the following categories:

  • Port API
  • Secure channel API
  • Secure association API
  • Packet flow and actions
  • Statistics
  • Event Handling API

MACSec as such is a Port feature and most of the Broadcom APIs are designed around that. Port APIs controls the MACSec features at the port level for example Port API will control whether or not the MACSec feature is enabled or not enabled for the port, so on. For the port that supports MACSec feature, it can have certain number of MACSec secure channels, Secure channel APIs provide the means to create/update/delete/Traverse secure channels on the port. For a specified secure channel, there can be certain number of secure Associations which belong to the Secure channel. MACSec secure association API provide the means to create/delete/update/Traverse MACSec secure Associations. Broadcom devices provide flexible packet parse engine which can be controlled though the MACSec flow APIs. These APIs provide the means to Create/Update/Delete/Traverse flows. These APIs can be used for classifying the traffic to either belong to MACSec controlled or uncontrolled port or to perform various actions like Dropping packets when hardware detects various error conditions in the packet etc.

Most of the MACSec API take unit and bcm_port_t parameters which identifies the port on BCM switch device.

MACSec Stats API provide access to the IEEE 802.1AE MIB for the specified port.

Secure MACSec Secure Channel API

The MACSec secure channel APIs provide the functions to create, update, delete and traverse MACSec secure channels on Broadcom devices. A secure channel is a software object which can be considered an abstraction of the MACSec secure channel. Secure channels are identified by a channel identifier (chanId). The chanId can be allocated either by the bcm_macsec_secure_chan_create API or a specific chanId to be used the channel. After the channel is created, use chanId for all subsequent secure channel APIs for this channel.

Once the channel is created, user should use chanId for all subsequent secure channel APIs for this channel.

API Brief
bcm_macsec_secure_chan_create(int unit, bcm_port_t port, uint32 flags, bcm_macsec_secure_chan_t* chan, int* chanId) Create a MACSec secure channel.
bcm_macsec_secure_chan_get(int unit, bcm_port_t port, int chanId, bcm_macsec_secure_chan_t* chan) Return MACSec secure channel information corresponding to channel identified by chanId.
bcm_macsec_secure_chan_destroy(int unit, bcm_port_t port, int chanId) Destroy MACSec secure channel corresponding to chanId.
bcm_macsec_secure_chan_traverse(int unit, bcm_port_t port, bcm_macsec_chan_traverse_cb calback, void* user_data) Traverse over all the secure channels for the specified port.
bcm_macsec_secure_assoc_create(int unit, bcm_port_t port, uint32 flags, int chanId, bcm_macsec_secure_assoc_t* assoc, int* assocId) Create a MACSec Secure association for the specified secure channel.
bcm_macsec_secure_assoc_get(int unit, bcm_port_t port, int assocId, bcm_macsec_secure_assoc_t* assoc, int* chanId) Return MACSec secure association for the specified assocId.
bcm_macsec_secure_assoc_destroy(int unit, bcm_port_t port, int assocId) Destroys MACSec secure association for the specified assocId.
bcm_macsec_secure_assoc_traverse(int unit, bcm_port_t port, int chanId, bcm_macsec_secure_assoc_traverse_cb cb, void* user_data) Traverse all the secure associations and call the user-provided callback function for each secure association.

Flow MACSec Flow API

Broadcom MACSec flow API provide the means of creating packet flows and assigning different actions to the flows. A flow is defined a the set of packets with a fixed set of values for different fields in the packet. These APIs program the field processor engine which does the packet lookup. Each flow has a corresponding action primitives that can be assigned to the flow. Once the packets matching the specified flow match signature are found, hardware takes the specified actions bound to that flow. There are 2 broad categories of flow i.e., Ingress flow and Egress flows. Egress flows are the ones going out from switch/ASIC towards the line side. Ingress flows are the ones coming from the line side and going towards the switch/ASIC.

API Brief
bcm_macsec_flow_create(int unit, bcm_port_t port, uint32 flags, bcm_macsec_flow_match_t* flow, bcm_macsec_flow_action_t* action, int* flowId) Create/update a MACSec flow object
bcm_macsec_flow_destroy(int unit, bcm_port_t port, int flowId) Destroy the MACSec flow object
bcm_macsec_flow_get(int unit, bcm_port_t port, int flowId, bcm_macsec_flow_match_t* flow, bcm_macsec_flow_action_t* a) Find and return the MACSec flow.
bcm_macsec_flow_traverse(int unit, bcm_port_t port, bcm_macsec_flow_traverse_cb callbk, void* user_data) Traverse all flows for the specified port and call user-provided callback function for each of the flows.

Event Event Handling API

Event handling API provides the application to receive notifications of certain events happening from the MACSec software. Most of the events occur in response to Hardware interrupts.

API Brief
bcm_macsec_event_register(int unit, bcm_macsec_event_cb cb, void* user_data) Register application event handler function
bcm_macsec_event_unregister(int unit, bcm_macsec_event_cb cb) Unregister the application event handler function.
bcm_macsec_event_enable_set(int unit, bcm_macsec_event_t t, int enable) Enable or disable event notification for the specified event.
bcm_macsec_event_enable_get(int unit, bcm_macsec_event_t t, int* enable) Get current event notification status for the specified event.
